# <a name="step-1-create-a-test-base-account"></a>Etapa 1: Criar uma conta da Base de Teste

Se você não tiver uma assinatura do Azure, crie uma [conta gratuita](https://azure.microsoft.com/en-us/free/) antes de começar.

## <a name="enter-details-for-test-base-account"></a>Insira detalhes da conta base de teste
 
1. Pesquise **'Base de Teste'** no portal do Azure.

![Criar uma imagem de pesquisa de Conta Base de Teste](Media/CreateTestAccount1.png)

2. Clique **em 'Adicionar'** para criar uma conta da Base de Teste.

![Clicando em adicionar para criar a conta](Media/CreateTestAccount2.png)

3.  Leia a caixa ```Terms of Use``` de seleção em seguida, para confirmar sua satisfação com ```Terms of Use``` o .

![Revisar os termos de uso](Media/CreateTestAccount3.png)

4.  Preencha as informações corretas nos seguintes requisitos: 
    -   Assinatura: Grupo de Recursos
    -   Detalhes da Instância: Nome.

**Atualmente, a Base de Teste oferece suporte apenas à camada Preço Padrão.**

![Selecione assinatura, grupo de recursos e digite os detalhes](Media/CreateTestAccount4.png)

5.  Por fim, clique em ```Review + Create``` Para validar e habilitar sua conta recém-criada.
