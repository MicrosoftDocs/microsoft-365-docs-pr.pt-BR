# <a name="support-for-microsoft-managed-desktop"></a>Suporte para a Área de Trabalho Gerenciada da Microsoft

Dispositivos de área de trabalho do Microsoft gerenciados são gerenciados por e serão reparados ou substituídos pela Microsoft quando necessário. Microsoft fornecerá o gerenciamento de incidentes proativo e reativo. Microsoft rastreia incidentes no portal de administração do Microsoft Desktop gerenciados. Eles são classificados de acordo com as [definições de gravidade](#severity-definitions). 

Os clientes podem contatar operações Microsoft Desktop gerenciados para:
- Solicitações de informações sobre o local de trabalho moderno como uma configuração ou Locatário de serviço
- Solicitações de alteração para a configuração do local de trabalho moderno como um serviço de dispositivos
- Relatórios de um incidente ou interrupção

## <a name="whats-included"></a>O que está incluído?

Suporte da Microsoft Desktop gerenciados:

- Fornece uma equipe de engenheiros de suporte dedicada ao dispositivos Microsoft Desktop gerenciados
- Oferece suporte a dispositivos Microsoft Desktop gerenciados e usuários finais atribuídos a esses dispositivos
- Concede limitado acesso administrativo ao dispositivos Microsoft Desktop gerenciados para engenheiros de suporte gerenciando dispositivos do Microsoft Desktop gerenciados 

Produtos suportados:

- Windows 10 com o Windows Defender avançadas de proteção de ameaça (WDATP) 
- Subconjunto do pacote Microsoft 365 E5: Outlook, Word, PowerPoint, Excel, Skype para cliente corporativos, Teams da Microsoft 
- Microsoft Store para Empresas 
- OneDrive para Business client 

Detalhes de suporte:

- Atual: América do Norte e do Reino Unido apenas (24 x 7 x 365) 
- Inglês é o idioma com suporte apenas para telefone e bate-papo conversas com os clientes 
- Podemos parceria com, não substituindo helpdesk de sua empresa; Aplicativos de linha de negócios (LOB), recursos de rede, etc. ainda são manipuladas pela assistência técnica 
- Dispositivos não-Microsoft Desktop gerenciados estão fora do escopo 

## <a name="hardware-replacement"></a>Substituição de hardware

Dispositivos de hardware são substituídos para os itens sob os termos da garantia selecionado. Dispositivos serão atualizados no final do termo seu dispositivo como serviço, como seus garantia anexado. Dispositivos que precisam ser substituído para motivos fora a cobertura de garantia será avaliada uma taxa de substituição, por exemplo, os dispositivos perdidos ou furtados. Para obter uma lista completa dos quais é abordada na garantia, consulte [Opções de garantia de superfície](https://support.microsoft.com/help/4036296/surface-surface-standard-warranty).
