# <a name="microsoft-managed-desktop-supported-regions"></a>Regiões com suporte da Área de Trabalho Gerenciada da Microsoft

Este artigo fornece detalhes sobre regiões onde você pode usar a Área de Trabalho Gerenciada da Microsoft e onde ela é suportada. Para obter mais informações sobre idiomas com suporte da Área de Trabalho Gerenciada da Microsoft para a experiência do usuário, consulte [Localize devices for users](../get-started/localization.md).

Você ainda pode usar dispositivos gerenciados fora dessas regiões sem interrupção para o serviço de Área de Trabalho Gerenciada da Microsoft. Por exemplo, um funcionário baseado no Reino Unido pode trabalhar com segurança e receber atualizações em seu dispositivo gerenciado durante a viagem para Ásia, Europa ou América do Sul.

Para obter mais informações sobre os idiomas suportados pela Área de Trabalho Gerenciada da Microsoft para experiências de usuário e administrador, consulte [Localize the user experience](../get-started/localization.md).

Para obter mais informações sobre o suporte do usuário com a Área de Trabalho Gerenciada da Microsoft, consulte [Support for Microsoft Managed Desktop](support.md).

## <a name="availability-of-the-service"></a>Disponibilidade do serviço

As organizações nos seguintes países podem se inscrever na Área de Trabalho Gerenciada da Microsoft:

- Estados Unidos
- Canadá (excluindo Quebec)
- Reino Unido
- Irlanda
- Bélgica
- Luxemburgo
- Países Baixos
- Suécia
- Finlândia
- Noruega
- Dinamarca
- Islândia
- Austrália
- Nova Zelândia
