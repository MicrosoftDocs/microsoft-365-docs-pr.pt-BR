# <a name="microsoft-managed-desktop-devices"></a>Dispositivos de área de trabalho gerenciada da Microsoft 

O Microsoft Managed desktop avalia regularmente os dispositivos que serão incluídos no serviço. Este tópico descreve os requisitos de um dispositivo para ser inscrito na área de trabalho gerenciada da Microsoft.

Para obter mais informações sobre a área de trabalho gerenciada da Microsoft, consulte [documentação e recursos do Microsoft Managed desktop](https://docs.microsoft.com/microsoft-365/managed-desktop/).

Além disso, o dispositivo também deve:     

- Ter uma imagem do Factory ou uma nova geração de imagens:
    - Tem a versão mais recente do Windows 10 pro ou Enterprise ou versão anterior imediata;
    - Tem a versão mais recente do Microsoft Office 365 clique para executar ou versão anterior imediata;
    - O tem os drivers mais recentes fornecidos pelo OEM instalados; e
    - Não tem software adicional pré-carregado
    - Dispositivos HP com 5VS04AV devem ter uma imagem de dispositivos gerenciados da Microsoft
- Não concluiu a experiência de primeira execução do Windows.
- Ser inscrito no [Windows AutoPilot](https://docs.microsoft.com/en-us/windows/deployment/windows-autopilot/user-driven-aad) e ter o perfil AutoPilot da área de trabalho gerenciada da Microsoft atribuído.
