# <a name="order-microsoft-managed-desktop-devices"></a>Solicitar Área de Trabalho Gerenciada da Microsoft dispositivos

Recomendamos trabalhar com um dos nossos parceiros de dispositivo aprovados. Você pode trabalhar com seu contato de conta da Microsoft para obter mais ajuda para configurar uma parceria de dispositivo.

Área de Trabalho Gerenciada da Microsoft não exige mais que os modelos de dispositivo sejam da lista de [dispositivos aprovados no momento.](../service-description/device-list.md) A partir de 3 de maio de 2021, todos os dispositivos de um fabricante aprovado devem atender aos nossos requisitos de hardware e software postados. Você pode continuar a usar dispositivos nessa lista com confiança, mas pode encontrar mais dispositivos recomendados para Área de Trabalho Gerenciada da Microsoft no site de dispositivos Windows 10 Pro [de](https://www.microsoft.com/windowsforbusiness/view-all-devices) negócios da Loja. Nesse site, veja os dispositivos recomendados expandindo **Recursos** na área **Filtro** por área e selecionando **Área de Trabalho Gerenciada da Microsoft**. Sempre que você planeja registrar um modelo de dispositivo específico no serviço pela primeira vez, você deve testar um exemplo para garantir que ele entregará a experiência do usuário esperada. Para obter mais informações, consulte [Validate new devices](../get-started/validate-device.md).
