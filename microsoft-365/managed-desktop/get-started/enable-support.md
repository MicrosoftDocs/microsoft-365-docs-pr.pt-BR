# <a name="enable-user-support-features"></a>Habilitar recursos de suporte ao usuário

Se você estiver fornecendo suporte ao seu próprio usuário ou trabalhando com um parceiro para fornecer, siga estas etapas para permitir que o provedor de suporte solicite acesso de dispositivos elevados ou a escalada de problemas para Área de Trabalho Gerenciada da Microsoft, se necessário.

1. Se eles ainda não têm uma, os usuários precisam de uma conta no mesmo domínio Azure Active Directory (AAD) que os Área de Trabalho Gerenciada da Microsoft dispositivos.
2. Adicione as contas de usuário da Etapa 1 ao grupo de segurança do Roles-Support Local de Trabalho **Moderno** no AAD.
