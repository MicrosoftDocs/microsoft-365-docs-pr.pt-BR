# <a name="get-your-users-ready-to-use-devices"></a>Prepare usuários para usarem os dispositivos

Quando um dispositivo de Área de Trabalho Gerenciada da Microsoft está nas mãos do seu usuário, começar a trabalhar com ele é rápido e fácil. Os dispositivos vêm pré-configurados com a versão atual do Windows, e as configurações e aplicativos são instalados a partir da nuvem à medida que o usuário conclui a configuração. 
 
Para facilitar ainda mais isso, oferecemos um guia que orienta seus usuários durante a configuração inicial e fornece recursos de ajuda para a instalação e para uso posterior, se necessário. Você pode personalizar este guia para incluir determinados detalhes específicos da sua organização. Em seguida, distribua o guia diretamente para os usuários juntamente com o dispositivo. Há duas versões, uma para organizações que usam o sistema de suporte da Microsoft (o aplicativo Obter Ajuda), uma para aquelas que estão fornecendo suporte alternativo para seus usuários.

## <a name="to-prepare-the-guide"></a>Para preparar o guia

1. Baixe a versão apropriada para sua organização:
- [Área de Trabalho Gerenciada](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-custom-v3.pdf) da Microsoft - Começar a usar seu dispositivo (versão obter Ajuda) - use essa versão se os usuários usarem o sistema de suporte da Microsoft.
- [Área de Trabalho Gerenciada](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-custom-v2.pdf) da Microsoft - Começar a usar seu dispositivo - use esta versão se os usuários não *usarem* o sistema de suporte da Microsoft.
2. Use qualquer aplicativo capaz de abrir arquivos PDF para preencher detalhes relevantes para sua organização:
    - O nome da rede à que os usuários devem se conectar para continuar a instalação (Etapa 3 no guia)
    - O nome da conta de locatário do Azure da sua organização (Etapa 4 no guia)
    - Informações de contato para o suporte de IT interno da sua organização (início da segunda página)
3. Salve o PDF editado e distribua aos usuários. 

## <a name="ready-to-use-guide"></a>"Guia pronto para uso"

Também fornecemos uma versão mais genérica do guia para as organizações que não precisam personalizá-lo. Há duas versões, uma para organizações que usam o sistema de suporte da Microsoft (o aplicativo Obter Ajuda), uma para aquelas que estão fornecendo suporte alternativo para seus usuários. 

- [Área de Trabalho](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-help-v3.pdf) Gerenciada da Microsoft - Começar a usar seu dispositivo (versão obter Ajuda - pronto para uso) - use essa versão se os usuários usarem o sistema de suporte da Microsoft.
- [Área de Trabalho Gerenciada](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/managed-desktop/get-started/downloads/microsoft-managed-desktop-user-guide-no-help-v2.pdf) da Microsoft - Começar a usar seu  dispositivo (pronto para uso) - use esta versão se os usuários não usarem o sistema de suporte da Microsoft.
