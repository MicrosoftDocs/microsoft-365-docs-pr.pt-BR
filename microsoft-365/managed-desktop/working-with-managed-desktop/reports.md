# <a name="work-with-reports"></a>Trabalho com relatórios

O Microsoft Managed Desktop fornece relatórios que administradores de ti em seu locatário podem usar para entender vários aspectos da população de dispositivos. Você acessa os relatórios no [portal de administração ](https://aka.ms/mmdportal)em **relatórios**.



## <a name="device-inventory-report"></a>Relatório do inventário do dispositivo

O modo de exibição **dispositivos** oferece uma guia **Exportar tudo** que [baixa um relatório detalhado de inventário](device-inventory-report.md).
