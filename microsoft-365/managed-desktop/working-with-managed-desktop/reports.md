# <a name="work-with-reports"></a>Trabalhar com relatórios

O Microsoft Managed Desktop fornece relatórios que administradores de ti em seu locatário podem usar para entender vários aspectos da população de dispositivos. Você acessa os relatórios no [portal de administração ](https://aka.ms/mmdportal)em **relatórios**.


## <a name="device-compliance-report"></a>Relatório de conformidade do dispositivo

O modo de exibição principal mostra a porcentagem de seus dispositivos de área de trabalho gerenciada da Microsoft que estão em conformidade com as políticas de conformidade definidas no Microsoft Intune. Você também pode exibir informações específicas de todos os dispositivos de área de trabalho gerenciado da Microsoft ou apenas dispositivos que não estão em conformidade com as políticas de conformidade e exportar um arquivo com as mesmas informações.

Saiba mais sobre o [relatório de conformidade do dispositivo](device-compliance-report.md).

## <a name="device-inventory-report"></a>Relatório de inventário de dispositivos

O modo de exibição **dispositivos** oferece uma guia **Exportar tudo** que [baixa um relatório detalhado de inventário](device-inventory-report.md).
