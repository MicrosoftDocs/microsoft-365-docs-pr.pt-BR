# <a name="device-inventory-report"></a>Relatório de inventário de dispositivos

No modo de exibição **dispositivos** , você pode selecionar a guia **Exportar tudo** para baixar um arquivo delimitado por vírgulas, incluindo estas informações:

- Número de série
- Nome do dispositivo
- Último check-in do dispositivo
- Gerenciado por
- Fabricantes
- Modelo
- Opera
- Licença de so
- Versão do sistema operacional
- Propriedade
- Data de registro
- Nome do toque
- Armazenamento livre em GB
- Armazenamento-total em GB
- Inscrito pelo UPN do usuário
- Usuário atribuído
- Status
- Idade (meses)
- Perfil
- Usuários conectados
