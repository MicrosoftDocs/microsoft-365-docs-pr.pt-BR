# <a name="device-inventory-report"></a>Relatório do inventário do dispositivo

Na **exibição Dispositivos,**  você pode selecionar a guia Exportar tudo para baixar um arquivo delimitado por vírgulas, incluindo essas informações:

- Número de série
- Nome do Dispositivo
- Último check-in do dispositivo
- Gerenciado por
- Fabricante
- Modelo
- Sistema operacional
- Licença do sistema operacional
- Versão do sistema operacional
- Propriedade
- Data do registro
- Nome do anel
- Armazenamento - Gratuito em GB
- Armazenamento - Total em GB
- Inscrito pelo UPN do usuário
- Usuário atribuído
- Status
- Idade (meses)
- Perfil
- Usuários conectados
