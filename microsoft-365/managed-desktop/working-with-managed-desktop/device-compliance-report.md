# <a name="device-compliance-report"></a>Relatório de conformidade do dispositivo

Este modo de exibição mostra a porcentagem de seus dispositivos de área de trabalho gerenciada da Microsoft que estão em conformidade com as políticas de conformidade definidas no Microsoft Intune.

![Relatório mostrando a porcentagem de dispositivos que estão em conformidade com as políticas](../../media/mmd-device-compliance-percent.png)


Selecione **Exibir detalhes** para exibir informações específicas de todos os dispositivos de área de trabalho gerenciado da Microsoft ou apenas dispositivos que não estejam em conformidade com as políticas de conformidade. Use o menu suspenso para alternar entre esses modos de exibição. Para aparecer nesse modo de exibição, um dispositivo deve ser uma máquina física e sincronizado com o Intune nos últimos 30 dias.

![Painel de detalhes com guias para todos os dispositivos ou dispositivos não compatíveis. Mostra o esquema detalhado, incluindo o número de série, o nome do dispositivo, o estado de conformidade e outras informações. Botão baixar dados completos na parte inferior.](../../media/mmd-device-compliance-detail.png)

Nesse modo de exibição, selecione **baixar dados completos** para exportar um arquivo delimitado por vírgula com as mesmas informações. No arquivo de exportação, as áreas exibidas como **esquema** aparecem como colunas.
