# <a name="work-with-insights"></a>Trabalhe com percepções

A Área de Trabalho Gerenciada da Microsoft fornece vários painéis que os administradores de IT em seu locatário podem usar para entender vários aspectos da população de dispositivos. Você pode acessá-los diretamente, no [Centro de administração do Microsoft 365.](https://admin.microsoft.com/adminportal/home?previewoff=false#/microsoftmanageddesktop)

Com esses painéis, você pode encontrar as respostas para perguntas como estas:

- Quantos dispositivos estão ativos e quando eles foram usados pela última vez?
- Quais aplicativos são mais usados e durante quais momentos?
- Quais aplicativos estão causando problemas falhando ou travando muito?
- Como as Operações de Área de Trabalho Gerenciada da Microsoft estão mitigando ou resolvendo esses problemas?
- Quais aplicativos estão consumindo mais energia?
- Qual é a duração prevista da bateria de meus dispositivos?
- Qual é o status atual das atualizações de segurança em dispositivos?
- Quanto tempo demora para que 95% dos dispositivos se atualizem com a atualização de segurança mais recente?


Para acessar essas exibições no [Microsoft Endpoint Manager,](https://endpoint.microsoft.com/)navegue até a guia Área de Trabalho Gerenciada da Microsoft na home page e selecione **Exibir** detalhes na **área Relatórios:**


![Página principal do Centro de administração com a área De relatórios no canto inferior esquerdo e link Exibir detalhes](../../media/insights-main.png)


## <a name="usage-insights"></a>Percepções de uso
Esta exibição fornece métricas de uso para seus dispositivos da Área de Trabalho Gerenciada da Microsoft. 

Para exibir dados de uso, selecione a **guia** Uso.

Saiba mais sobre as [informações de uso.](usage-insights.md)

## <a name="reliability-insights"></a>Percepções de confiabilidade
Esta exibição fornece um resumo de saúde de seus dispositivos gerenciados. Para exibir dados de confiabilidade, selecione a **guia** Confiabilidade.

Saiba mais sobre [percepções de confiabilidade.](reliability-insights.md)

## <a name="battery-insights"></a>Percepções da bateria
Esta exibição mostra informações sobre o consumo de energia de aplicativos e duração projetada da bateria para dispositivos em seu ambiente. Para exibir essas informações, selecione a **guia** Bateria.

Saiba mais sobre [percepções de bateria.](battery-insights.md)

## <a name="windows-security-update-insights"></a>Percepções de atualização da segurança do Windows
Esta exibição mostra informações sobre o status das atualizações de segurança dos dispositivos da Área de Trabalho Gerenciada da Microsoft. Para exibir essas informações, selecione a **guia Atualizações de Segurança do Windows.**

Saiba mais sobre as [informações de atualização de segurança.](security-update-insights.md)
