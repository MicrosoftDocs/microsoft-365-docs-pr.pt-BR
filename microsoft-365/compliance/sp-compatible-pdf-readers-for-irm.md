# <a name="sharepoint-compatible-pdf-readers-that-support-microsoft-information-rights-management-services"></a>SharePoint-Compatible pdf que suportam serviços de Gerenciamento de Direitos de Informação da Microsoft

O Microsoft SharePoint 2013 oferece suporte à proteção do IRM (Gerenciamento de Direitos de Informação) de documentos PDF. Com esse suporte, os usuários podem carregar documentos PDF em bibliotecas protegidas por IRM e, ao baixar, os arquivos serão protegidos usando Microsoft Office IRM.
  
Para usar arquivos PDF em bibliotecas que o proprietário protegeu com IRM, o usuário precisará obter um dos seguintes leitores compatíveis com PDF:
