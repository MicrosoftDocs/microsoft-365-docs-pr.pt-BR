# <a name="download-selected-documents-from-a-review-set"></a>Baixar documentos selecionados de um conjunto de revisão

O download oferece uma maneira simples de baixar conteúdo de um conjunto de revisão no formato nativo. A ferramenta de download Advanced eDiscovery aproveita os recursos de transferência de dados do navegador. Um prompt de navegador será exibido quando um download estiver pronto. Os arquivos baixados usando esse método são cortados em um arquivo contêiner e conterão arquivos no nível do item. Isso significa que, se você selecionar baixar um anexo, receberá a mensagem de email com o anexo incluído. Da mesma forma, se você exportar uma planilha Excel que está inserida em um documento do Word, o documento do Word com a planilha Excel incorporada será incluído no download. Quando você baixou itens, a propriedade Last Modified Data é preservada e pode ser exibida como uma propriedade de arquivo.

Para baixar o conteúdo de um conjunto de revisão em Advanced eDiscovery caso, comece selecionando os arquivos que você deseja baixar e selecione **Itens de ação**  >  **Baixar**.
