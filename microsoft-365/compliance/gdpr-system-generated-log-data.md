# <a name="additional-steps-to-export-system-generated-log-data"></a>Etapas adicionais para exportar dados de log gerados pelo sistema

**Kaizala**&nbsp;&nbsp;&nbsp;O portal de gerenciamento Kaizala permite que você exporte dados de uso de produtos e serviços da organização e depois use funcionalidades do Excel para filtrar os dados de um usuário específico. Para obter instruções detalhadas, confira [Exportar dados organizacionais no Kaizala](https://support.office.com/article/export-user-data-in-kaizala-150ec214-a070-4e8a-8509-82f46d84bbb6).

**Serviço de Roaming do Office**&nbsp;&nbsp;&nbsp;Roaming do Office é um serviço que armazena configurações relacionadas ao Office, como temas, dicionário personalizado, configurações de idioma, modo do desenvolvedor e correção automática do Office. Para obter instruções sobre como exportar esses dados, confira [Exportar dados do Serviço de Roaming do Office](https://support.office.com/article/manage-gdpr-data-subject-requests-with-the-dsr-case-tool-in-the-office-365-security-compliance-center-preview-ce9eb942-3589-42cb-88fd-1576ecb09c5c?storagetype=stage#o365datamoreinfo). 
 
**Workplace Analytics**&nbsp;&nbsp;&nbsp;A ferramenta de exportação do log de dados fornece os dados de uso para os usuários de sua organização que têm permissão para executar relatórios do Workplace Analytics. Além disso, o Workplace Analytics calcula e armazena dados com pseudônimo derivados de dados do Office 365 para melhorar o desempenho. Se quiser disponibilizar esses dados com pseudônimo para um usuário e precisar de assistência, contate o [Suporte da Microsoft](https://support.microsoft.com/contactus/).

**Yammer**&nbsp;&nbsp;&nbsp;O centro de administração do Yammer permite exportar dados de atividade da conta do usuário. Ao exportar os dados do usuário, você receberá uma mensagem de email que contém os dados de atividade da conta do usuário. Forneça essas informações ao usuário se quiser. Para obter instruções detalhadas, confira [Yammer Enterprise: privacidade](https://support.office.com/article/eae49f12-4661-4ba5-aa72-01248f0709bf).
