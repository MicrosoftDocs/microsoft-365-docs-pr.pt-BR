# <a name="data-subject-requests-for-the-gdpr"></a>Solicitações de Entidades de Dados para o RGPD

O RGPD (Regulamento Geral sobre a Proteção de Dados) fornece direitos às pessoas (conhecidas na regulamentação como entidades de dados) para gerenciar os dados pessoais que foram coletados por um funcionário ou outro tipo de órgão ou organização (conhecido como controlador de dados ou apenas controlador). Os dados pessoais são definidos muito amplamente no RGPD como quaisquer dados relacionados a uma pessoa identificada ou identificável. O RGPD fornece às entidades de dados direitos específicos a seus dados pessoais; esses direitos incluem a obtenção de cópias deles, a solicitação de alterações neles, a restrição do processamento deles, a exclusão deles ou seu recebimento em formato eletrônico para que possam ser migrados para outro controlador. Uma solicitação formal por uma entidade de dados a um controlador para realizar uma ação quanto a seus dados pessoais é chamada de DSR ou Solicitação de Entidade de Dados. O controlador é obrigado a considerar prontamente cada DSR e oferecer uma resposta substantiva executando a ação solicitada ou fornecendo uma explicação sobre por que o DSR não pode ser realizado pelo controlador. Um controlador deve consultar seus próprios consultores jurídicos ou de conformidade sobre a disposição adequada de qualquer DSR.

Estes artigos descrevem como usar produtos, serviços e ferramentas administrativas da Microsoft para ajudá-lo a encontrar e agir em relação a dados pessoais para responder a DSRs:

- [Office 365](gdpr-dsr-Office365.md)
- [Azure](gdpr-dsr-Azure.md)
- [Intune](gdpr-dsr-Intune.md)
- [Dynamics 365](gdpr-dsr-Dynamics365.md)
- [Família do Visual Studio](gdpr-dsr-visual-studio-family.md)
- [Azure DevOps Services](gdpr-dsr-vsts.md)
- [Suporte e Serviços Profissionais da Microsoft](gdpr-dsr-prof-services.md)

Para obter mais informações sobre como a Microsoft habilita você a responder a DSRs, confira [RGPD: DSRs (Solicitações de Entidades de Dados)](https://servicetrust.microsoft.com/ViewPage/GDPRDSR) no Portal de Confiança do Serviço.
