## <a name="restore-inactive-mailboxes"></a><span data-ttu-id="b19be-127">Restaurar caixas de correio inativas</span><span class="sxs-lookup"><span data-stu-id="b19be-127">Restore inactive mailboxes</span></span>

<span data-ttu-id="b19be-128">Use o cmdlet **New-MailboxRestoreRequest** com os parâmetros  _SourceMailbox_ e  _TargetMailbox_ para restaurar o conteúdo de uma caixa de correio inativa para uma caixa de correio existente.</span><span class="sxs-lookup"><span data-stu-id="b19be-128">Use the **New-MailboxRestoreRequest** cmdlet with the  _SourceMailbox_ and  _TargetMailbox_ parameters to restore the contents of an inactive mailbox to an existing mailbox.</span></span> <span data-ttu-id="b19be-129">Para obter mais informações sobre como usar esse cmdlet, consulte [New-MailboxRestoreRequest](/powershell/module/exchange/new-mailboxrestorerequest).</span><span class="sxs-lookup"><span data-stu-id="b19be-129">For more information about using this cmdlet, see [New-MailboxRestoreRequest](/powershell/module/exchange/new-mailboxrestorerequest).</span></span>

   ```powershell

   ```

   > [!IMPORTANT]
