# <a name="compliance-solutions"></a>Soluções de conformidade
Essa biblioteca fornece recursos técnicos para o uso de recursos no Microsoft 365 a fim de trabalhar para a conformidade com as regulamentações que se aplicam a muitas organizações, começando pelo [GDPR](gdpr.md). 

Enquanto adicionamos conteúdo à biblioteca, não deixe de examinar os recursos de conformidade na Central de Confiabilidade da Microsoft:
- [Visão geral de conformidade](https://www.microsoft.com/trustcenter/compliance)
- [Ofertas de conformidade](https://www.microsoft.com/trustcenter/compliance/complianceofferings)
