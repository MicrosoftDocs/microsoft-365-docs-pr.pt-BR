### <a name="introduction-to-dpias"></a><span data-ttu-id="f0a4b-110">Introdução ao DPIAs</span><span class="sxs-lookup"><span data-stu-id="f0a4b-110">Introduction to DPIAs</span></span>

<span data-ttu-id="f0a4b-p103">Sob o gerais dados proteção regulamentação (GDPR), controladores de dados são necessários para preparar um DPIA para processamento de operações que são "poderá resultar em um alto risco para os direitos e a liberdade de pessoas naturais." Não há nada inerentes serviços profissionais Microsoft que necessariamente exigiriam a criação de um DPIA por um controlador de dados utilizá-lo. Rather, whether a DPIA is required will be dependent on the details and context of the type of services and how the data controller uses the professional services.</span></span>

<span data-ttu-id="f0a4b-114">A finalidade deste documento é fornecer controladores de dados com informações sobre os serviços profissionais que ajudarão a determinar se é necessário um DPIA e, em caso afirmativo, o que detalhes para incluir.</span><span class="sxs-lookup"><span data-stu-id="f0a4b-114">The purpose of this document is to provide data controllers with information about Professional Services that will help them to determine whether a DPIA is needed and, if so, what details to include.</span></span>
