# <a name="get-started-with-the-microsoft-service-trust-portal"></a>Começar a trabalhar com o Portal de Confiança do Serviço da Microsoft

O Portal de Confiança do Serviço da Microsoft fornece uma variedade de conteúdo, ferramentas e outros recursos sobre as práticas de segurança, privacidade e conformidade da Microsoft.

## <a name="accessing-the-service-trust-portal"></a>Acessando o Portal de Confiança do Serviço

O Portal de Confiança do Serviço contém detalhes sobre a implementação de controles e processos da Microsoft que protegem nossos serviços de nuvem e os dados do cliente neles. Para acessar alguns dos recursos no Portal de Confiança do Serviço, você deve entrar como um usuário autenticado com sua conta de serviços de nuvem da Microsoft (uma conta de organização do Azure Active Directory ou uma conta da Microsoft) e analisar e aceitar o Contrato de Não Divulgação da Microsoft para Materiais de Conformidade.

### <a name="existing-customers"></a>Clientes existentes

Os clientes existentes podem acessar o Portal de Confiança do Serviço <https://aka.ms/STP> em uma das seguintes assinaturas online (avaliação ou paga):

- Microsoft 365
- Dynamics 365
- Azure

 > [!NOTE]
 > As contas do Azure Active Directory associadas às organizações têm acesso à gama completa de documentos e recursos, como o Gerenciador de Conformidade. As contas da Microsoft criadas para uso pessoal têm acesso limitado ao conteúdo do Portal de Confiança do Serviço.

### <a name="new-customers-and-customers-evaluating-microsoft-online-services"></a>Novos clientes e clientes avaliando os serviços online da Microsoft

Para criar uma nova conta ou criar uma conta de avaliação, use um dos seguintes formulários de inscrição (também usados para contas de avaliação) para obter acesso ao STP.

- Inscreva-se para uma nova conta de avaliação do [Microsoft 365 Apps para](https://go.microsoft.com/fwlink/p/?LinkID=507653) Empresas ou uma nova conta de avaliação do Office [365 Enterprise](https://go.microsoft.com/fwlink/p/?LinkID=698279)

- Inscreva-se em uma nova conta [de avaliação do Dynamics 365](https://go.microsoft.com/fwlink/?LinkId=252780)

- Inscreva-se em uma nova [conta de avaliação do Azure.](https://go.microsoft.com/fwlink/?LinkId=722737)

Ao se inscrever para uma avaliação gratuita ou uma assinatura, você deve habilitar o Azure Active Directory para dar suporte ao seu acesso ao STP.

## <a name="using-the-service-trust-portal"></a>Usando o Portal de Confiança do Serviço

Os recursos e o conteúdo do Portal de Confiança do Serviço podem ser acessados no menu principal.

![Menu do Portal de Confiança do Serviço](../media/STPMenus1.png)

As seções a seguir descrevem cada item no menu principal.

### <a name="service-trust-portal"></a>Portal de Confiança do Serviço

O link **Portal de Confiança do** Serviço exibe a home page. Ele fornece uma maneira rápida de voltar à home page.

### <a name="compliance-manager"></a>Gerente de Conformidade

> [!IMPORTANT]
> O Gerenciador de Conformidade foi movido do Portal de Confiança do Serviço para seu novo local no centro de conformidade [do Microsoft 365.](https://compliance.microsoft.com/) Todos os dados do cliente foram transferidos para o novo local, para que você possa continuar usando o Gerenciador de Conformidade sem interrupção. Consulte a [documentação do Gerenciador de Conformidade para](compliance-manager.md) obter informações de configuração e saber mais sobre os novos recursos. Embora a versão clássica do Gerenciador de Conformidade permaneça no Portal de Confiança do Serviço, todos os usuários são incentivados a usar o Gerenciador de Conformidade no centro de conformidade do Microsoft 365.

### <a name="trust-documents"></a>Documentos de Confiança

Fornece uma grande quantidade de informações de implementação e design de segurança com o objetivo de facilitar o cumprimento dos objetivos de conformidade regulamentar entendendo como os serviços do Microsoft Cloud mantêm seus dados seguros. Para revisar o conteúdo, selecione uma  das seguintes opções no menu suspenso Documentos de Confiança.

- **Relatórios de Auditoria:** Uma lista de relatórios independentes de auditoria e avaliação nos serviços de nuvem da Microsoft é exibida. Esses relatórios fornecem informações sobre a conformidade dos serviços do Microsoft Cloud com os padrões de proteção de dados e requisitos regulatórios, como:

  - ISO (International Organization for Standardization)
  - Service Organization Controls (SOC)
  - NIST (Instituto Nacional de Padrões e Tecnologia)
  - FedRAMP (Programa Federal de Gerenciamento de Autorização e Risco)
  - Regulamento Geral de Proteção de Dados (RGPD)

- **Proteção de dados:** Contém uma grande quantidade de recursos, como controles auditados, white papers, perguntas frequentes, testes de penetração, ferramentas de avaliação de risco e guias de conformidade.

- **Blueprints de Segurança e Conformidade do Azure:** Recursos que ajudam a criar aplicativos seguros e compatíveis baseados em nuvem. Essa área contém diretrizes de projetos para o governo, finanças, assistência médica e verticais de varejo.

### <a name="industries--regions"></a>Países e regiões

Fornece informações de conformidade específicas do setor e da região sobre os serviços do Microsoft Cloud.

- **Industries:** No momento, esta página fornece uma página de aterrissagem específica do setor para o setor de Serviços Financeiros. Isso contém informações como ofertas de conformidade, perguntas frequentes e histórias de sucesso. Os recursos para mais setores serão lançados no futuro, no entanto, você pode encontrar recursos para mais setores indo para a página Documentos de Confiança **>** Proteção de Dados no STP.

- **Regiões:** Fornece opiniões legais sobre a conformidade dos serviços do Microsoft Cloud com várias leis de vários países. Países específicos incluem Austrália, Canadá, República Tcheca, Dinamarca, Alemanha, Polônia, Romênia, Espanha e Reino Unido.

### <a name="trust-center"></a>Central de Confiabilidade

Links para a [Central de Confiações da Microsoft,](https://www.microsoft.com/trust-center)que fornece mais informações sobre segurança, conformidade e privacidade no Microsoft Cloud. Isso inclui informações sobre os recursos nos serviços do Microsoft Cloud que você pode usar para atender aos requisitos específicos do RGPD, documentação útil para sua responsabilidade do RGPD e para o entendimento das medidas técnicas e organizacionais que a Microsoft tomou para dar suporte ao RGPD.

### <a name="my-library"></a>Minha Biblioteca

Esse novo recurso permite salvar *(ou* fixar) documentos para que você possa acessá-los rapidamente na página Minha Biblioteca. Você também pode configurar notificações para que a Microsoft envie uma mensagem de email quando os documentos em sua Biblioteca são atualizados. Para obter mais informações, consulte [a seção Minha](#my-library-1) Biblioteca neste artigo.

### <a name="more"></a>Mais

Vá para **Mais > Administrador** para acessar funções administrativas que só estão disponíveis para a conta de administrador global. Essa opção fica visível somente quando você está londo como um administrador global. Há duas opções no menu **suspenso** Admin:

- **Configurações:** Esta página permite atribuir funções de usuário para o Gerenciador de Conformidade (clássico).

- **Configurações de privacidade do usuário:** Esta página permite exportar um relatório que contém atribuições de item de ação no Gerenciador de Conformidade (clássico) para um usuário específico. Você também pode reatribuir todos os itens de ação a um usuário diferente e remover qualquer item de ação atribuído do usuário especificado.

### <a name="search"></a>Pesquisar

Clique na lupa no canto superior direito da página do Portal de Confiança do Serviço para expandir a caixa, insira os termos de pesquisa e pressione **Enter.** A **página** Pesquisa é exibida, com o termo de pesquisa exibido na caixa de pesquisa e os resultados da pesquisa listados abaixo.

![Portal de Confiança do Serviço – pesquisa em documentos com filtro aplicado](../media/86b754e1-c63c-4514-89ac-d014bf334140.png)

Por padrão, a pesquisa retorna resultados do documento. Você pode filtrar os resultados usando as listas suspensos para refinar a lista de documentos exibidos. Você pode usar vários filtros para restringir a lista de documentos. Os filtros incluem os serviços de nuvem específicos, categorias de práticas de conformidade ou segurança, regiões e setores. Clique no link do nome do documento para baixar o documento.

Para listar controles de Avaliações no Gerenciador de Conformidade (clássico) relacionados aos termos de pesquisa, clique em **Gerenciador de Conformidade.** Os resultados da pesquisa mostram a data em que a avaliação foi criada, o nome do grupo de avaliação, o serviço microsoft Cloud aplicável e se o controle é gerenciado pela Microsoft ou pelo cliente. Clique no nome do controle para exibir o controle na Avaliação no Gerenciador de Conformidade (clássico).

![Portal de Confiança do Serviço – pesquisa em controles de gerenciamento de conformidade](../media/bafb811a-68ce-40b5-ad16-058498fe5439.png)

> [!NOTE]
> Relatórios e documentos do Portal de Confiança do Serviço estão disponíveis para download por pelo menos 12 meses após a publicação ou até que uma nova versão do documento fique disponível.

## <a name="my-library"></a>Minha Biblioteca

Use o recurso Minha Biblioteca para adicionar documentos e recursos no Portal de Confiança do Serviço à sua página Minha Biblioteca. Isso permite acessar documentos relevantes para você em um único local.  Para adicionar um documento à sua Biblioteca, clique no **menu ...** à direita de um documento e selecione Salvar **na biblioteca.** Você pode adicionar vários documentos à sua Minha Biblioteca clicando na caixa de  seleção ao lado de um ou mais documentos e clicando em Salvar na biblioteca na parte superior da página.

Além disso, o recurso de notificações permite que você configure sua Minha Biblioteca para que uma mensagem de email seja enviada a você sempre que a Microsoft atualizar um documento que você adicionou à sua Biblioteca. Para configurar notificações, vá para Sua Biblioteca e clique em **Configurações de Notificação.** Você pode escolher a frequência de notificações e especificar um endereço de email em sua organização para o qual enviar notificações. As notificações por email incluem links para os documentos que foram atualizados e uma breve descrição da atualização.

Observe também que identificamos todos os documentos em sua Biblioteca que foram atualizados nos últimos 30 dias, independentemente de você ativar ou não as notificações. Uma breve descrição da atualização também é exibida em uma dica de ferramenta.

## <a name="starter-packs"></a>Pacotes de início

Os pacotes de início são um conjunto de documentação da Microsoft sobre os serviços do Microsoft Cloud para setores específicos. Atualmente, o Portal de Confiança do Serviço oferece os três pacotes de início a seguir para organizações de serviços financeiros. Esses pacotes de início ajudam as organizações a avaliar a segurança, a conformidade e a privacidade no Microsoft Cloud e fornecem orientações para ajudar a implementar os serviços de nuvem da Microsoft no setor de serviços financeiros altamente regulamentados.

- **Pacote de Início da Avaliação:** Use para avaliação antecipada da nuvem da Microsoft para organizações de serviços financeiros.

- **Pacote de Início da Avaliação:** Após a avaliação, use as listas de verificação e outras diretrizes neste pacote inicial para ajudar sua organização a avaliar os riscos relacionados à segurança, conformidade e privacidade.

- **Pacote de Início de Auditoria:** Use este pacote inicial para obter orientação sobre como usar controles de auditoria e outras ferramentas para ajudar a orientar a implementação dos serviços do Microsoft Cloud de uma maneira que ajude a reduzir a exposição a riscos da sua organização.

To access these starter packs, go to **Service Trust Portal > Industries & Regions > Industry Solutions > Financial Services**. Você pode abrir ou baixar documentos de um pacote inicial ou salvá-los em sua Minha Biblioteca.
