# <a name="compliance-manager-templates-list"></a>Lista de modelos do Gerenciador de Conformidade

**Neste artigo:** Exibir a lista abrangente de **modelos disponíveis** para criar avaliações no Gerenciador de Conformidade.

> [!IMPORTANT]
> Os modelos de avaliação que estão disponíveis para sua organização dependem do contrato de licenciamento. [Revise os detalhes](compliance-manager-templates.md#template-availability-and-licensing).

## <a name="overview"></a>Visão Geral

[O Microsoft Compliance Manager](compliance-manager.md) fornece um conjunto abrangente de modelos para criar avaliações. Esses modelos podem ajudar sua organização a cumprir requisitos nacionais, regionais e específicos do setor que regem a coleta e o uso de dados.

Os modelos são adicionados ao Gerenciador de Conformidade à medida que novas leis e regulamentos são aprovados. O Gerenciador de Conformidade também atualiza seus modelos quando as leis ou regulamentos subjacentes mudam. Saiba mais sobre como revisar [e aceitar atualizações.](compliance-manager-assessments.md#accept-updates-to-assessments)

## <a name="list-of-templates-and-where-to-find-them"></a>Lista de modelos e onde encontrá-los

Abaixo está a lista completa de modelos no Gerenciador de Conformidade. Os links nos nomes de modelo abaixo levam você à documentação relacionada sobre esse padrão, regulamentação ou lei.

### <a name="where-to-find-your-templates"></a>Onde encontrar seus modelos

Para revisar os modelos disponíveis para sua organização, vá para a página Modelos **de Avaliação.** Saiba mais sobre [como exibir e gerenciar seus modelos.](compliance-manager-templates.md#view-and-manage-templates)

## <a name="included-templates"></a>Modelos incluídos
Um ou mais desses modelos estarão disponíveis com base no contrato de licenciamento. O modelo de Linha de Base de Proteção de Dados está incluído para todos os usuários.

- [Linha de base da Proteção de Dados da Microsoft](compliance-manager-assessments.md#data-protection-baseline-default-assessment)
- [RGPD da União](/compliance/regulatory/gdpr) Europeia (Microsoft 365, Office 365, Intune)
- [ISO/IEC 27001:2013](/compliance/regulatory/offering-iso-27001)
- NIST 800-53 Revs. 4 e 5

> [!NOTE]
> Para clientes do Community (GCC) do Governo dos EUA, moderados, do GCC Alto e do Departamento de Defesa (DoD), os modelos CMMC (Certificação do Modelo de Maturidade da Segurança Cibernética) níveis 1 a 5 estão incluídos, além dos modelos listados acima.
