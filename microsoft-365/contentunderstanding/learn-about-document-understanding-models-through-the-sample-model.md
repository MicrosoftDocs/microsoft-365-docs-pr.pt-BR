# <a name="learn-about-document-understanding-models-through-a-sample-model"></a>Saiba mais sobre documentos entendendo modelos por meio de um modelo de exemplo

O Microsoft SharePoint Syntex fornece um modelo de exemplo que você pode usar para examinar, dando a você uma compreensão melhor de como criar seus próprios modelos. O modelo de amostra também permite examinar componentes de modelo, como classificador, extratores e explicações. Você também pode usar os arquivos de exemplo para treinar o modelo.

## <a name="import-the-sample-model"></a>Importar o modelo de amostra

Para acessar o modelo de amostra, você precisa primeiro importar o modelo para o seu centro de conteúdo.

1. No centro de conteúdo, selecione **modelos** para ver sua lista de modelos.</br>
2. Na página **modelos** , selecione **modelo de amostra de importação**.</br>

    ![Importar modelo de amostra](../media/content-understanding/import-sample-model.png) </br>

3. Procure o modelo de exemplo intitulado *BenefitsChangeNotice. classificador* que aparece na sua lista de modelos.</br>

    ![Modelo de amostra](../media/content-understanding/sample-model.png) </br>

4. Selecione o *BenefitsChangeNotice. classificador*, para abrir a home page do modelo.</br>
  
     ![Página inicial de exemplo](../media/content-understanding/sample-home-page.png)

5. Na home page do modelo, examine o modelo mais atentamente para ver como ele foi criado.
 
- Exibir os arquivos de exemplo usados para treinar o modelo.
- Examine os extratores de entidade do modelo mais atentamente para ver como o modelo de exemplo configurou as explicações.

   ![Extratores de modelo de amostra](../media/content-understanding/entity-extractors.png)
