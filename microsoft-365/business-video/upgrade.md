# <a name="upgrade-from-windows-10-home-to-windows-10-pro"></a>Atualizar do Windows 10 Home para o Windows 10 Pro

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE3t58j?autoplay=false]

## <a name="try-it"></a>Experimente!

O Microsoft 365 Business exige que o Windows 10 Pro ajude a proteger seus dados. Se você tiver um computador executando o Windows 10 Home, siga estas etapas para atualizar para o Windows 10 Pro.

1. Na área de trabalho, escolha o menu  **Iniciar,**  procure  **a Microsoft Store** e selecione-o.
2. Na Microsoft Store, clique na barra de pesquisa, digite  **Windows 10 Pro** e escolha  **Atualizar para o Windows 10 Pro**.
3. Em  **Atualizar para o Windows 10 Pro,** escolha Saber  **mais** e escolha  **Comprar**.
4. Na caixa  **Segurança do Windows,**  insira seu PIN para verificar sua identidade.
5. Revise a página de pagamento, leia os termos de venda, faça alterações no método de pagamento, se quiser, e escolha  **Comprar**.
6. Em  **Atualizar para o Windows 10 Pro,** escolha  **Instalar**.
7. Salve todos os arquivos, se necessário, e escolha  **Sim,&#39;vamos**  começar a instalar o Windows.
