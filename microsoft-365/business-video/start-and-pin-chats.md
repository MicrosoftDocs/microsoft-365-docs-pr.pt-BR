# <a name="chat-with-employees-and-customers"></a>Converse com funcionários e clientes

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RWedV6?autoplay=false]

Você pode bater papo internamente com seus colegas de trabalho ou bater papo com um usuário convidado, como um cliente ou parceiro. Se o [acesso de convidados estiver ativado](https://docs.microsoft.com/microsoftteams/set-up-guests) para sua empresa, você pode simplesmente digitar um endereço de email para adicionar um convidado a um chat.

## <a name="try-it"></a>Experimente!

Inicie os chats de 1:1 ou de grupo para colaborar com um grupo menor ou para conversar de forma privada e fixar chats para acesso rápido.

1. Em  **bate-papos** , você pode ter 1:1 ou agrupar chats com pessoas.
2. Selecione **novo chat** :::image type="icon" source="../media/new-chat-icon.png" border="false"::: .  
3. Digite o nome da pessoa ou das pessoas que você deseja contatar.
4. Selecione a seta para baixo para adicionar um nome ao seu grupo de chat.

Para fazer uma chamada de vídeo ou áudio de um chat

- Selecione chamada de  **vídeo** :::image type="icon" source="../media/video-call-icon.png" border="false"::: ou **chamada** :::image type="icon" source="../media/audio-call-icon.png" border="false"::: de áudio para ligar para alguém diretamente de um chat.
