# <a name="create-a-website"></a>Criar um site

> [!VIDEO https://www.microsoft.com/videoplayer/embed/4839abc6-9323-4cbf-a79d-2907235f9ebb?autoplay=false]

## <a name="try-it"></a>Experimente!

O Microsoft 365 não inclui um site público para sua empresa. Para configurar uma, considere usar uma empresa de hospedagem na Web, como GoDaddy, Wix ou WordPress.

1. Procure um provedor de hospedagem na Web onde você possa criar seu site.

     Você pode achar mais fácil escolher o construtor de sites na mesma empresa em que seu nome de domínio está registrado.
1. Projete seu site de negócios, certifique-se de incluir o logotipo da empresa, a finalidade da empresa, o texto, as imagens e os links.
1. Para integrar o site com o Microsoft 365, você pode adicionar seu email comercial ao site e um link para o Microsoft Teams, o SharePoint ou outra ferramenta de colaboração.
1. Certifique-se de conectar seu site ao seu nome de domínio, que você&#39;também está usando para email no Microsoft 365.
