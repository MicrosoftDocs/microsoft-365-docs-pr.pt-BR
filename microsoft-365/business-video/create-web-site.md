# <a name="create-a-website"></a>Criar um site

> [!VIDEO https://www.microsoft.com/videoplayer/embed/4839abc6-9323-4cbf-a79d-2907235f9ebb?autoplay=false]

## <a name="try-it"></a>Experimente!

O Microsoft 365 não inclui um site público para a sua empresa. Para configurar um, considere usar uma empresa de hospedagem na Web como GoDaddy, WiX ou WordPress.

1. Procure um provedor de hospedagem na Web em que você pode criar seu site.

     Talvez seja mais fácil escolher o criador de sites na mesma empresa onde o nome de seu domínio está registrado.
1. Projete seu site de negócios, certificando-se de incluir seu logotipo de negócios, objetivo de negócios, texto, imagens e links.
1. Para integrar o site com o Microsoft 365, você pode adicionar seus emails comerciais ao site e um link para o Microsoft Teams, SharePoint ou outra ferramenta de colaboração.
1. Certifique-se de conectar seu site ao nome de domínio, que você&#39;reutilizar também para email no Microsoft 365.
