# <a name="install-office-apps"></a>Instalar os aplicativos do Office 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/acce002c-0756-4b64-ac5d-2198ee96a9b1?autoplay=false]

## <a name="try-it"></a>Experimente!

Microsoft Office aplicativos podem ser encontrados no **menu** Iniciar. Se você não&#39;vê-los, pode instalá-los por conta própria.

1. Vá para office.com. Talvez seja necessário entrar com sua conta de trabalho.
2. Selecione **Instalar Office** Microsoft 365   >      >   **aplicativos Executar** e selecione **Sim**.
3. Os Office aplicativos serão instalados. O processo pode levar vários minutos. Quando ele for concluído, selecione  **Fechar**.
4. Para instalar Microsoft Teams, vá para a página office.com e escolha **Teams**.
5. Obter o Windows e, em seguida, selecione **Executar**. Teams exibe um prompt quando a instalação é concluída.

Os Office que você instalou agora aparecem no **menu** Iniciar.
