# <a name="evaluate-microsoft-defender-antivirus"></a>Avaliar o Microsoft Defender Antivírus

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Aplica-se a:**

- [Microsoft Defender para Ponto de Extremidade](/microsoft-365/security/defender-endpoint/)

Use este guia para determinar a Microsoft Defender Antivírus protege contra vírus, malware e aplicativos potencialmente indesejados.

>[!TIP]
>Você também pode visitar o site de demonstração do Microsoft Defender para Ponto de Extremidade no [demo.wd.microsoft.com](https://demo.wd.microsoft.com?ocid=cx-wddocs-testground) para confirmar que os seguintes recursos estão funcionando e ver como eles funcionam:
>- Proteção fornecida na nuvem
>- Aprendizado rápido (incluindo Bloquear à primeira vista)
>- Bloqueio de aplicativo potencialmente indesejado

Ele explica os recursos importantes de proteção de próxima geração Microsoft Defender Antivírus disponíveis para pequenas e grandes empresas e como eles aumentam a detecção e a proteção de malware em toda a sua rede.

Você pode optar por configurar e avaliar cada configuração independentemente ou tudo de uma vez. Agrupamos configurações semelhantes com base em cenários de avaliação típicos e incluímos instruções para usar o PowerShell para habilitar as configurações.

O guia está disponível no formato PDF para exibição offline:

- [Baixar o guia no formato PDF](https://www.microsoft.com/download/details.aspx?id=54795)

Você também pode baixar um PowerShell que habilita todas as configurações descritas no guia automaticamente. Você pode obter o script juntamente com o download em PDF acima ou individualmente na Galeria do PowerShell:

- [Baixe o script do PowerShell para configurar automaticamente as configurações](https://www.powershellgallery.com/packages/WindowsDefender_InternalEvaluationSettings)

> [!IMPORTANT]
> No momento, o guia destina-se à avaliação de máquina única Microsoft Defender Antivírus. A habilitação de todas as configurações neste guia pode não ser adequada para implantação no mundo real.
>
> Para saber mais sobre as recomendações mais recentes para implantação e monitoramento do Microsoft Defender Antivírus em uma rede, consulte [Deploy Microsoft Defender Antivírus](deploy-manage-report-microsoft-defender-antivirus.md).
