# <a name="check-the-microsoft-defender-for-endpoint-service-health"></a>Verificar a saúde do serviço do Microsoft Defender para Ponto de Extremidade

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]


**Aplica-se a:**
- [Microsoft Defender para Ponto de Extremidade](https://go.microsoft.com/fwlink/?linkid=2154037)



>Deseja experimentar o Microsoft Defender para Ponto de Extremidade? [Inscreva-se para uma avaliação gratuita.](https://www.microsoft.com/microsoft-365/windows/microsoft-defender-atp?ocid=docs-wdatp-servicestatus-abovefoldlink)

**A saúde** do serviço fornece informações sobre o status atual do serviço Defender para Ponto de Extremidade. Você poderá verificar se a saúde do serviço está saudável ou se há problemas atuais. Se houver problemas, você verá informações como quando o problema foi detectado, qual é a causa raiz preliminar e o tempo de resolução esperado.

Você também verá informações sobre problemas históricos que foram resolvidos e detalhes como a data e a hora em que o problema foi resolvido. Quando não houver problemas no serviço, você verá um status saudável.

Você pode exibir detalhes sobre a saúde do  serviço clicando no painel operações de segurança ou selecionando o menu **de** saúde do serviço no painel de navegação.

A **página Detalhes de** saúde do serviço tem as seguintes guias:

- **Status atual**
- **Histórico de status**

## <a name="current-status"></a>Status atual
A **guia Status** atual mostra o estado atual do serviço Defender para Ponto de Extremidade. Quando o serviço está em execução suavemente, uma saúde de serviço saudável é mostrada. Se houver problemas, os seguintes detalhes do serviço serão mostrados para ajudá-lo a obter informações sobre o problema:

- Data e hora para quando o problema foi detectado
- Uma breve descrição do problema
- Hora da atualização
- Resumo do impacto
- Causa raiz preliminar
- Próximas etapas
- Tempo de resolução esperado

As atualizações sobre o andamento de um problema são refletidas na página à medida que o problema é resolvido. Você verá atualizações sobre informações como um tempo de resolução de estimativa atualizado ou próximas etapas.

Quando um problema é resolvido, ele é gravado na **guia Histórico de** status.

## <a name="status-history"></a>Histórico de status
A **guia Histórico de** Status reflete todos os problemas históricos que foram vistos e resolvidos. Você verá detalhes dos problemas resolvidos juntamente com as outras informações que foram incluídas enquanto estavam sendo resolvidas.
