# <a name="next-generation-protection"></a>Proteção de última geração

[!INCLUDE [Microsoft 365 Defender rebranding](../../includes/microsoft-defender.md)]

**Aplica-se a:**

- [Microsoft Defender para Ponto de Extremidade](/microsoft-365/security/defender-endpoint/)

## <a name="microsoft-defender-antivirus-your-next-generation-protection"></a>Microsoft Defender Antivírus: sua proteção de última geração

O Microsoft Defender Antivírus é o componente de proteção de última geração do Microsoft Defender para Ponto de Extremidade. Essa proteção reúne aprendizado de máquina, análise de big data, pesquisa aprofundada de resistência a ameaças e a infraestrutura de nuvem da Microsoft para proteger os dispositivos da sua organização. Os serviços de proteção de última geração incluem os seguintes recursos:

- [Proteção antivírus baseada em comportamento, heurística e em tempo real](configure-protection-features-microsoft-defender-antivirus.md), que inclui verificação constante usando monitoramento de comportamento de arquivos e processos e outras heurísticas (também conhecida como *proteção em tempo real*). Também inclui a detecção e o bloqueio de aplicativos considerados não seguros, mas que podem não ser detectados como malware.
- [Proteção fornecida pela nuvem](cloud-protection-microsoft-defender-antivirus.md), que inclui detecção e bloqueio quase instantâneo de ameaças novas e emergentes.
- [Proteção dedicada e atualizações de produto](manage-updates-baselines-microsoft-defender-antivirus.md), que incluem atualizações relacionadas a manter o Microsoft Defender Antivírus atualizado.

## <a name="try-a-demo"></a>Experimente uma demonstração!

Visite o [site de demonstração do Microsoft Defender para Ponto de Extremidade](https://demo.wd.microsoft.com?ocid=cx-wddocs-testground) para confirmar se os seguintes recursos de proteção estão funcionando e para explorá-los usando cenários de demonstração:
- Proteção fornecida na nuvem
- Proteção de bloqueio à primeira vista (BAFS)
- Proteção contra aplicativos potencialmente indesejados (PUA)
