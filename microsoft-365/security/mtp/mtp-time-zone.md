# <a name="set-the-time-zone-for-microsoft-365-defender"></a>Definir o fuso horário para o Microsoft 365 defender

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Aplica-se a:**
- Microsoft 365 defender



O Microsoft 365 defender pode exibir informações de data e hora usando o fuso horário local ou o UTC. O fuso horário escolhido será aplicado a todas as informações de data e hora exibidas nos seguintes recursos no centro de segurança do Microsoft 365:
- Incidentes
- Investigação e correção automatizadas, incluindo a central de ação
- Busca avançada

Para definir o fuso horário desses recursos, vá para **configurações** de  >  fuso horário do **Microsoft 365 defender**  >  **Time zone**.
