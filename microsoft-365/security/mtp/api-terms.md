## <a name="apis"></a>APIs

 As APIs do Microsoft 365 defender são regidas pela [licença e termos de uso da API da Microsoft](https://docs.microsoft.com/legal/microsoft-apis/terms-of-use).

### <a name="throttling-limits"></a>Redução dos limites

Cada API tem seus próprios limites de limitação.<br>
Há limites de limitação por hora e por minuto.<br>
Por exemplo, os limites de limitação da API de incidentes são: 50 chamadas por minuto e 1500 por hora.<br>
O código de status de resposta HTTP, em caso de limitação, será de 429.
