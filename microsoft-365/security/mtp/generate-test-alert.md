# <a name="generate-a-test-alert-in-your-microsoft-365-defender-evaluation-lab"></a>Gerar um alerta de teste no laboratório de avaliação do Microsoft 365 defender  

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender.md)]


**Aplica-se a:**
- Microsoft 365 defender

Agora que você concluiu a instalação e a configuração do laboratório de avaliação do Microsoft 365 defender, é possível executar uma simulação de ataque de ameaça para experimentar a detecção, a correlação de alerta e os recursos de remediação de auto-recuperação da solução de segurança integrada.  

1. Faça logon no https://security.microsoft.com
2. Na Home Page, procure uma **simulação** com título de bloco.  Revise o guia passo a passo para obter as etapas para criar um incidente de teste no Microsoft 365 defender.
<br>![Imagem do cartão de simulação no painel de segurança do Microsoft 365](../../media/mtp-eval-73.png) <br>
