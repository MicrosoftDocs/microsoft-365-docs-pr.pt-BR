# <a name="generate-a-test-alert-in-your-microsoft-threat-protection-evaluation-lab"></a>Gerar um alerta de teste em seu laboratório de avaliação de proteção contra ameaças da Microsoft  

**Aplica-se a:**
- Proteção contra Ameaças da Microsoft

Agora que você concluiu a instalação e a configuração do laboratório de avaliação de proteção contra ameaças da Microsoft, é possível executar uma simulação de ataque de ameaça para experimentar a detecção, a correlação de alerta e os recursos de correção de auto-recuperação da solução de segurança integrada.  

1. Faça logon no https://security.microsoft.com
2. Na Home Page, procure uma **simulação**com título de bloco.  Revise o guia de orientações sobre as etapas para criar um incidente de teste no Microsoft Threat Protection.
