# <a name="microsoft-threat-protection-prerequisites"></a>Pré-requisitos da proteção contra ameaças da Microsoft

**Aplica-se a:**
- Proteção contra Ameaças da Microsoft

[!INCLUDE [Prerelease information](../includes/prerelease.md)]

Saiba mais sobre os requisitos de licenciamento, hardware e software, além de outras configurações para executar e usar a segurança do Microsoft 365.

## <a name="licensing-requirements"></a>Requisitos de licença
A segurança do Microsoft 365 exige uma das seguintes licenças:

- Microsoft 365 E5 
- Office 365 E5, Enterprise Mobility + Security E5, e Windows E5

Você pode adquirir essas licenças na [página do Microsoft 365 Enterprise](https://www.microsoft.com/en-us/microsoft-365/enterprise).

### <a name="check-your-existing--licenses"></a>Verificar suas licenças existentes
Acesse o centro de administração do Microsoft 365 em [admin.microsoft.com](https://admin.microsoft.com/) para exibir suas licenças existentes. No Centro de administração, acesse **Cobrança** > **Licenças**.

É necessário atribuir a função do **Administrador de cobranças** ou do **Leitor global** [no Azure AD](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#available-roles) para ver as informações de licenciamento. Se tiver problemas de acesso, entre em contato com um administrador global.  

## <a name="browser-requirements"></a>Requisitos de navegador
O acesso à central de segurança do Microsoft 365 é feito por meio de um navegador. O Internet Explorer e o Microsoft Edge são compatíveis. Todos os navegadores compatíveis com o HTML5 também são compatíveis.
