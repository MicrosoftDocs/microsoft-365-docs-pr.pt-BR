# <a name="sending-mail-to-microsoft-365"></a>Enviando emails para o Microsoft 365

[!INCLUDE [Microsoft 365 Defender rebranding](../includes/microsoft-defender-for-office.md)]


Esses artigos ajudam os envios externos a melhorar sua reputação e a aumentar a capacidade de entregar emails aos usuários no Microsoft 365. Eles também fornecem algumas informações sobre como você pode relatar tentativas de lixo eletrônico e phishing, mesmo que você não seja um usuário do Microsoft 365.

Se você não for um cliente, mas estiver tentando enviar emails para alguém que está, você está no lugar certo. Se você for um administrador e precisar de ajuda para combater o spam, essa não é a seção certa para você. Em vez disso, vá para [Proteção anti-spam e anti-malware no Microsoft 365](anti-spam-and-anti-malware-protection.md).
