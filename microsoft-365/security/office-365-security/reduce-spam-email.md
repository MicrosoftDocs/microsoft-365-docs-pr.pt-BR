## <a name="fixing-allowed-spam"></a><span data-ttu-id="a2e2d-110">Corrigir spam permitido</span><span class="sxs-lookup"><span data-stu-id="a2e2d-110">Fixing allowed spam</span></span>

### <a name="for-admins"></a><span data-ttu-id="a2e2d-118">Para administradores</span><span class="sxs-lookup"><span data-stu-id="a2e2d-118">For admins</span></span>

