<span data-ttu-id="acbba-150">O trecho a seguir de um arquivo CSL mostra um conjunto abrangente de orientações marcadas como comentários com `//` .</span><span class="sxs-lookup"><span data-stu-id="acbba-150">The following excerpt from a CSL file shows a comprehensive set of guidance marked as comments with `//`.</span></span>

```kusto

// Contains
// - Account information associated with the logon
// - The device which the account logged onto
// - The process which performed the logon
// - Network information (for network logons)
// - Timestamp
```

```kusto
DeviceLogonEvents
| count

// DeviceLogonEvents
```
