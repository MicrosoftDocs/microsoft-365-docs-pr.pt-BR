# <a name="apply-for-tax-exempt-status-for-office-365-for-business"></a>Candidatar-se ao status de isento de imposto para o Office 365 para empresas

Se você estiver qualificado para obter o status de isenção de imposto em seu país ou região, [entre em contato](../../admin/contact-support-for-business-products.md) com o suporte para criar uma solicitação de serviço para estabelecer o status de isenção de imposto para sua assinatura. Você precisará fornecer a seguinte documentação ao enviar sua solicitação: 
  
|**Localização**|**Documentação**|
|:-----|:-----|
|Estados Unidos e Canadá  <br/> |Certificado de imposto sobre uso de vendas  <br/> |
|Porto Rico  <br/> |Certificado de compras exentas  <br/> |
|Irlanda  <br/> |Certificado de isenção de imposto 13B/56A  <br/> |
|Organizações internacionais que mantêm a isenção de impostos  <br/> |Confirmação de certificação/carta de autoridades de impostos locais relevantes  <br/> |
