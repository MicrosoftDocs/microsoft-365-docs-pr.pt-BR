# <a name="billing-information-for-microsoft-365-for-business-in-mexico"></a>Informações de cobrança para o Microsoft 365 for Business no México

## <a name="1-where-can-i-get-an-invoice-for-my-microsoft-365-for-business-purchase"></a>1. onde posso obter uma fatura para minha compra da Microsoft 365 para empresas?

Sua fatura será enviada para o endereço de email de notificação de cobrança registrado 24 horas após a confirmação da compra. Para baixar sua fatura no centro de administração do Microsoft 365, vá para a página **cobrança** de  >  <a href="https://go.microsoft.com/fwlink/p/?linkid=2102895" target="_blank">contas & pagamentos</a> .

> [!IMPORTANT]
> Se você deseja que a fatura inclua sua RFC, você deve adicioná-la antes da confirmação de compra.

## <a name="2-how-can-i-add-my-rfc-registro-federal-de-contribuyentes-to-the-invoice-i-get-for-the-purchase-of-microsoft-365-for-business"></a>2. como posso adicionar minha RFC (registro Federal de Contribuyentes) à fatura, recebo para a compra do Microsoft 365 for Business?

### <a name="during-the-purchase-process-checkout"></a>Durante o processo de compra (check-out)

Durante o processo de compra de check-out, quando você chegar à etapa **4. Pagamento e cobrança**, selecione a caixa para inserir uma RFC. Essa etapa é necessária para que você possa vê-la refletida na fatura de compra.

### <a name="for-your-future-purchases"></a>Para suas compras futuras

Você pode adicionar ou modificar sua RFC para que as informações sejam refletidas nas futuras compras da Microsoft 365 para empresas. Para adicionar ou modificar sua RFC, acesse a **Billing**  >  página <a href="https://go.microsoft.com/fwlink/p/?linkid=2084771" target="_blank">contas de cobrança</a> de cobrança.

> [!NOTE]
> Se você estiver em uma assinatura de cobrança mensal ou anual, a adição ou a modificação da RFC será refletida na fatura de sua próxima cobrança.

## <a name="3-can-i-add-or-modify-my-rfc-to-an-invoice-that-was-already-generated"></a>3. posso adicionar ou modificar minha RFC a uma fatura que já foi gerada?

É importante que você adicione ou modifique sua RFC antes de fazer sua compra. Após gerarmos sua fatura do Microsoft 365 para empresas, você não poderá adicionar ou modificar as informações de RFC.

## <a name="4-why-dont-i-get-a-cfdi-comprobante-fiscal-digital-por-internet"></a>4. por que não recebo uma CFDI (Comprobante fiscal digital por Internet)?

A Microsoft Corporation, como o imposto residente nos Estados Unidos da América, só é capaz de emitir faturas sob as leis de jurisdição e não é obrigada a fornecer uma fatura eletrônica do CFDI.

## <a name="5-is-the-invoice-i-receive-valid-for-my-accounting-operations-in-mexico"></a>5. a fatura que recebo é válida para minhas operações contábeis no México?

Uma fatura gerada por entidades estrangeiras (neste caso, Microsoft Corporation) que inclui a respectiva RFC é válida para fins tributários no México. (Recomendamos consultar seu contador para obter mais informações).
