# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>Perguntas frequentes sobre o programa de uso do Microsoft Home

Os clientes de licenciamento por volume usam o programa Microsoft Home use (HUP) no [site da Microsoft Store para empresas](https://go.microsoft.com/fwlink/?linkid=2139192). 

Todos os outros clientes acessam o HUP usando esta etapa:

- No centro de administração da Microsoft, vá para a página **cobrança**  >  [**de seus produtos** ](https://go.microsoft.com/fwlink/p/?linkid=842054), selecione a guia **benefícios** e, em seguida, selecione a guia **HUP** .

## <a name="what-is-the-microsoft-home-use-program"></a>O que é o programa Microsoft Home use?

O programa Microsoft Home use (HUP) permite que os funcionários de empresas qualificadas comprem produtos Microsoft Consumer por um desconto. Você pode verificar os descontos atuais para cada oferta de produto visitando a [página de produtos do programa de uso doméstico](https://www.microsoft.com/home-use-program).

## <a name="in-which-countries-is-this-benefit-available"></a>Em que países esse benefício está disponível?

Depois de ativar seu benefício aqui (na Microsoft Store para empresas), os descontos do HUP nos produtos da Microsoft estão disponíveis para funcionários na maioria dos países em todo o mundo, com a exceção de:

- Rússia
- Venezuela

Os descontos do HUP também podem estar disponíveis em compras de dispositivos de superfície e acessórios e passagem de jogos do Xbox. Para disponibilidade, confira [Microsoft 365 subscriptions](https://go.microsoft.com/fwlink/p/linkid=2139192).

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>Como uma organização está qualificada para descontos do HUP?

Para que os funcionários aproveitem o desconto do programa de uso inicial da Microsoft, eles precisam de um endereço de email corporativo válido e de uma conta ativa da Microsoft, e sua organização deve atender a um dos seguintes critérios:

- Clientes que têm o Active Software Assurance para qualificar aplicativos do Office 

- Clientes que têm o Microsoft 365/Microsoft 365 E3/E5 do Software Assurance

- Clientes comerciais e governamentais que adquiriram licenças do ≥ 2.000 de uma ou uma combinação de Microsoft 365 E3/e5

- Clientes educacionais e não lucrativos

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>Por que preciso criar uma conta do Azure Active Directory (AAD) para usar o programa de uso doméstico/Microsoft Store para empresas?

Para atualizar e modernizar o programa de uso doméstico para uma variedade maior de benefícios de licença e produto, precisávamos alterar a forma como cumprimos o HUP. A Microsoft Store para empresas é usada para validação de qualificação e para atribuir benefícios do HUP e para que ele se baseia no AAD para conectar informações do VSLC e validar os domínios de email nos quais os funcionários da sua organização estão. Para obter mais informações, consulte [para inscrever-se nas contas do Azure ad por meio do Office 365 para empresas](https://docs.microsoft.com/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome).

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>Como posso se tornar o administrador de ti atribuído de um locatário gerenciado?

Você deve criar um registro especial no DNS para provar que é proprietário do domínio da sua empresa, por exemplo,:contoso. com.

Para obter um processo passo a passo para se tornar um locatário gerenciado, consulte [assumir um diretório não gerenciado como administrador no Azure active](https://docs.microsoft.com/azure/active-directory/users-groups-roles/domains-admin-takeover).

## <a name="where-can-i-find-marketing-materials-to-help-promote-the-program-within-my-company"></a>Onde posso encontrar materiais de marketing para ajudar a promover o programa dentro da minha empresa?

Você pode encontrar materiais de marketing de integração da empresa na seção _sobre_ do seu portal de administração, sob o link material de integração do programa de uso do Microsoft Home.

Você também pode localizar rapidamente materiais promocionais na página recursos de administração de ti/RH. Visite a [Home Page do programa de uso doméstico](https://www.microsoft.com/home-use-program/resources) e role para baixo até a seção _recursos promocionais do programa de uso doméstico_ da página.

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>Quais recursos estão disponíveis para ajudar a promover o benefício do programa de uso doméstico para os funcionários?

Use os recursos localizados na home page do [programa usar](https://www.microsoft.com/home-use-program/resources) para ajudá-lo a promover o programa para seus funcionários. Você encontrará panfletos digitais e imprimíveis, argumentação para uso em emails e boletins informativos e orientações sobre como promover melhor o programa de uso doméstico.

Você pode acessar o material de integração em vários idiomas no [HUP material para os clientes](https://microsofteur.sharepoint.com/teams/HUPMaterial). Selecione um idioma para começar a baixar os arquivos. O arquivo contém ativos que você pode usar para ajudar a ensinar seus funcionários sobre o programa e promover internamente em sua empresa.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>Quais são as etapas de um funcionário qualificado para fazer com que um programa de uso inicial da Microsoft compre?

1. **Inscreva-se** Vá para a [página de programa de uso inicial da Microsoft](https://www.microsoft.com/home-use-program)e selecione o botão introdução. Os funcionários inserem seu endereço de email corporativo.

2. **Confirmar**  Um email é enviado para a caixa de entrada do endereço de email corporativo com um link que autentica sua qualificação para o programa.

3. **Entrar** Depois de terem sido autenticados, eles começarão o processo de compra fornecendo a conta da Microsoft que está associada ao seu endereço de email pessoal, o que permite que eles exibam os descontos nos produtos disponíveis com base no benefício do programa de uso doméstico da organização. Para obter mais informações, consulte [como entrar em sua conta da Microsoft](https://support.microsoft.com/help/4028195/microsoft-account-sign-in).

4. **Shop** A compra é processada e concluída na Microsoft Store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Se um funcionário inscrever-se no Microsoft 365 por meio do programa Microsoft Home use, como isso afeta o limite de dispositivo e a assinatura de negócios do Microsoft 365 da organização?

Instala através do programa de uso doméstico não tem impacto na assinatura de negócios do Microsoft 365 da sua organização. Quando os funcionários ativam o Microsoft 365 em seus dispositivos pessoais, eles usam sua conta pessoal da Microsoft, e não suas contas de trabalho. Os administradores de ti da sua organização não gerenciam nem dão suporte a compras feitas por meio do programa de uso doméstico.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>Estou tendo problemas para gerenciar o benefício do programa Microsoft Home da minha organização. Onde posso obter suporte?

Para obter suporte para problemas de migração do centro de serviços de licenciamento por volume ou para acessar a Microsoft Store para empresas, consulte [suporte do VLSC](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0).

Para problemas de exibição do benefício após você já ter acessado a Microsoft Store para empresas, consulte [Store for Business admin support](https://docs.microsoft.com/microsoft-store/)

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>Por que não vejo uma opção para comprar o Visio Professional 2019 e/ou o Project Professional 2019?

Se um dos seus funcionários tiver aproveitado o programa anterior e estiver procurando descontos no Visio Professional 2019 e/ou no Project Professional 2019, passe pelo processo de validação de qualificação e, após entrar com sua conta pessoal da Microsoft, vá para **outros produtos do Office** para comprar esses produtos. Se você não vir os descontos esperados aplicados, entre em contato com o administrador do programa de casa do empregador para perguntar sobre esses benefícios adicionais.

## <a name="what-if-my-employees-have-questions-about-this-program-or-need-support-with-purchasing-and-installation"></a>E se meus funcionários tiverem dúvidas sobre esse programa ou se precisarem de suporte para compra e instalação?

Se os funcionários têm dúvidas, consulte os seguintes recursos:

- [Perguntas frequentes sobre compra e instalação](https://products.office.com/microsoft-office-for-home-and-school-faq)

- [Perguntas frequentes sobre funcionários do programa de uso doméstico](https://www.microsoft.com/home-use-program/frequently-asked-questions)

 Para obter ajuda com problemas técnicos, consulte [suporte da Microsoft](https://support.microsoft.com/). Você encontrará ajuda para:

- Status da ordem

- Cobrança

- Instalando o Office

- Entrar no Office
