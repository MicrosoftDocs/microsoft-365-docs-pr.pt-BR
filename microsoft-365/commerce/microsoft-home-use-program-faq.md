# <a name="microsoft-home-use-program-frequently-asked-questions-faq"></a>Perguntas frequentes sobre o Programa de Uso Home da Microsoft (FAQ)

Os clientes de Licenciamento por Volume usam o Programa de Uso Home da Microsoft (HUP) no site [da Microsoft Store para Empresas.](https://go.microsoft.com/fwlink/?linkid=2139192) 

Todos os outros clientes acessam o HUP usando esta etapa:

- No Centro de administração da Microsoft, vá para a página Cobrança de seus produtos, selecione a guia Benefícios e, em seguida, selecione a  >  [](https://go.microsoft.com/fwlink/p/?linkid=842054) **guia HUP.** 

## <a name="what-is-the-microsoft-home-use-program"></a>O que é o Programa microsoft Home Use?

O Microsoft Home Use Program (HUP) permite que os funcionários de empresas qualificadas comprem produtos de consumidor da Microsoft com um desconto. Você pode verificar os descontos atuais para cada oferta de produto visitando a página de produtos do Programa de Uso [Inicial.](https://www.microsoft.com/home-use-program)

## <a name="in-which-markets-is-the-home-use-program-offered"></a>Em quais mercados o Programa de Uso Doméstico é oferecido?

Os descontos em assinaturas do Microsoft 365 Family and Personal são oferecidos a empresas qualificadas em todo o mundo com estas exceções:

- Macau, Rússia e Venezuela: esta oferta não está disponível nesses mercados.
- China e Hong Kong: uma assinatura de um ano está disponível nesses mercados. Para a China, [visite o Microsoft 365 na China.](https://www.microsoftstore.com.cn/home-use-program/invite) Para Hong Kong, visite [Comprar o Microsoft 365 por um preço com desconto.](https://www.microsoftestore.com.hk/partner/hup?locale=en_HK)
- Japão: a Família microsoft 365 não está disponível.

## <a name="how-does-an-organization-qualify-for-hup-discounts"></a>Como uma organização se qualifica para descontos HUP?

Para que os funcionários aproveitem o desconto do Programa Microsoft Home Use, eles precisam de um endereço de email corporativo válido e uma conta da Microsoft ativa, e sua organização deve atender a um dos seguintes critérios:

- Clientes que têm Software Assurance ativo para qualificar aplicativos do Office 

- Clientes que têm o Microsoft 365/Microsoft 365 E3/E5 do Software Assurance

- Clientes comerciais e governamentais que compraram ≥ 2.000 licenças de uma ou uma combinação do Microsoft 365 E3/E5

- Clientes sem fins lucrativos e sem fins lucrativos

## <a name="why-do-i-have-to-create-an-azure-active-directory-aad-account-to-use-the-home-use-program-microsoft-store-for-business"></a>Por que preciso criar uma conta do Azure Active Directory (AAD) para usar o Programa home use/ Microsoft Store para Empresas?

Para atualizar e modernizar o Programa de Uso Home para uma ampla variedade de licenças e benefícios do produto, precisamos alterar a maneira como atendemos o HUP. A Microsoft Store para Empresas é usada para validação de qualificação e para atribuir benefícios de HUP e, por isso, depende do AAD para conectar informações do VSLC e validar os domínios de email nos quais os funcionários da sua organização estão. Para saber mais, confira Para se inscrever em contas do [Azure AD por meio do Office 365 para empresas.](https://docs.microsoft.com/microsoft-store/sign-up-microsoft-store-for-business#o365-welcome)

## <a name="how-do-i-become-the-assigned-it-admin-of-a-managed-tenant"></a>Como me tornar o administrador de IT atribuído de um locatário gerenciado?

Você deve criar um registro especial no DNS para provar que você é o próprio domínio da sua empresa, ex:contoso.com.

Para um processo passo a passo para se tornar um locatário gerenciado, confira Assumir um diretório não gerenciado como administrador no [Azure Active.](https://docs.microsoft.com/azure/active-directory/users-groups-roles/domains-admin-takeover)

## <a name="what-resources-are-available-to-help-promote-the-home-use-program-benefit-to-employees"></a>Quais recursos estão disponíveis para ajudar a promover o benefício do Programa de Uso Home aos funcionários?

Use os recursos localizados na home page do [Programa Home Use](https://www.microsoft.com/home-use-program/resources) para ajudá-lo a promover o programa para seus funcionários. Você encontrará folhetos digitais e imprimíveis, verbiage para uso em emails e boletins informativos e orientações sobre como promover melhor o Programa de Uso Acessível.

Você pode acessar materiais de integração em vários idiomas no [material HUP para clientes.](https://microsofteur.sharepoint.com/teams/HUPMaterial) Selecione um idioma para começar a baixar os arquivos. O arquivo contém ativos que você pode usar para ajudar a instruir seus funcionários sobre o programa e a promova internamente em sua empresa.

## <a name="what-are-the-steps-for-an-eligible-employee-to-make-a-microsoft-home-use-program-purchase"></a>Quais são as etapas para um funcionário qualificado fazer uma compra do Programa Microsoft Home Use?

1. **Inscrever-se** Vá para a [página do Programa Microsoft Home Use](https://www.microsoft.com/home-use-program)e selecione o botão Começar. Os funcionários insere seu endereço de email corporativo.

2. **Confirmar**  Um email é enviado para a caixa de entrada do endereço de email corporativo com um link que autentica sua qualificação para o programa.

3. **Entrar** Depois de autenticados, eles iniciam o processo de compra fornecendo a conta da Microsoft associada ao endereço de email pessoal, que permite que eles vejam os descontos dos produtos disponíveis com base no benefício do Programa de Uso Base da organização. Para obter mais informações, [consulte Como entrar em sua conta da Microsoft.](https://support.microsoft.com/help/4028195/microsoft-account-sign-in)

4. **Comprar** A compra é processada e concluída na Microsoft Store.

## <a name="if-an-employee-subscribes-to-microsoft-365-through-the-microsoft-home-use-program-how-does-this-impact-our-organizations-microsoft-365-business-subscription-and-device-limit"></a>Se um funcionário assina o Microsoft 365 por meio do Programa Microsoft Home Use, como isso afeta a assinatura comercial e o limite de dispositivos da nossa organização?

As instalações por meio do Programa de Uso Base não têm impacto na assinatura comercial do Microsoft 365 da sua organização. Quando os funcionários ativam o Microsoft 365 em seus dispositivos pessoais, eles usam a conta pessoal da Microsoft, não a conta comercial. Os administradores de IT da sua organização não gerenciam ou suportam as compras feitas por meio do Programa de Uso Home.

## <a name="im-having-trouble-managing-my-organizations-microsoft-home-use-program-benefit-where-can-i-get-support"></a>Estou com problemas para gerenciar o benefício do Programa Microsoft Home Use da minha organização. Onde posso obter suporte?

Para suporte para problemas de migração do Centro de Serviços de Licenciamento por Volume ou para acessar a Microsoft Store para Empresas, consulte [Suporte do VLSC.](https://www.microsoft.com/Licensing/servicecenter/default.aspx?wa=wsignin1.0)

Para problemas de visualização do benefício depois de já ter acessado a Microsoft Store para Empresas, consulte [Suporte para Administradores da Microsoft Store para Empresas](https://docs.microsoft.com/microsoft-store/)

## <a name="why-am-i-not-seeing-an-option-to-purchase-visio-professional-2019-andor-project-professional-2019"></a>Por que não vejo uma opção para comprar o Visio Professional 2019 e/ou o Project Professional 2019?

Se um dos seus funcionários se aproveitar do programa anterior e estiver procurando descontos no Visio Professional 2019 e/ou no Project Professional 2019, vá para o processo de validação de qualificação e, depois de entrar com sua conta pessoal da Microsoft, vá para Outros produtos do **Office** para comprar esses produtos. Se você não vir os descontos esperados aplicados, entre em contato com o administrador do Programa de Uso Home do seu empregador para saber mais sobre esses benefícios adicionais.

## <a name="what-if-my-employees-have-questions-about-this-program-or-need-support-with-purchasing-and-installation"></a>E se meus funcionários têm dúvidas sobre esse programa ou precisam de suporte para compra e instalação?

 Para ajuda com problemas técnicos, consulte Suporte [da Microsoft.](https://support.microsoft.com/) Você encontrará ajuda para:

- Status do pedido

- Cobrança

- Instalando o Office

- Entrar no Office
