## <a name="tenant-id-filtering-and-tenant-id-term-prefixing"></a><span data-ttu-id="8448c-121">Filtragem de ID de Locatário e Prefixação de Termos de ID de Locatário</span><span class="sxs-lookup"><span data-stu-id="8448c-121">Tenant ID Filtering and Tenant ID Term Prefixing</span></span>
