# <a name="microsoft-365-client-app-support-single-sign-on"></a>Microsoft 365 Suporte ao aplicativo cliente: single Sign-On

*Esse artigo se aplica ao Microsoft 365 Enterprise e ao Office 365 Enterprise.*

O SSO (sign-on único) adiciona segurança e conveniência quando os usuários se inssoam em aplicativos Azure Active Directory. Com o logor único, os usuários se ingressam uma vez com uma conta para acessar dispositivos ingressados no domínio do Active Directory (AD DS) locais, aplicativos saaS (serviço) e aplicativos Web.

Saiba mais sobre [o login único](/azure/active-directory/manage-apps/what-is-single-sign-on).

## <a name="supported-clients--platforms"></a>Clientes com suporte & plataformas

As versões mais recentes dos seguintes clientes e plataformas suportam logon único. Para obter mais informações sobre o suporte à plataforma Microsoft 365, consulte [System requirements for Microsoft 365](/microsoft-365/microsoft-365-and-office-resources).
