# <a name="plan-for-microsoft-365-enterprise"></a>Planejar o Microsoft 365 Enterprise

Quando você move uma organização corporativa para o Microsoft 365, é importante planejar com antecedência e tomar as principais decisões de design que simplificarão a implantação de TI e a adoção do usuário. 

## <a name="planning-with-microsoft-365-fasttrack"></a>Planejamento com o Microsoft 365 FastTrack

[O FastTrack para Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) é o melhor método para obter ajuda da Microsoft para planejar sua implantação do Microsoft 365. O FastTrack pode ajudá-lo com as considerações de design mais comuns e pode responder perguntas ao longo do caminho. 

>[!Note]
>Você também pode obter ajuda de um [parceiro da Microsoft.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-planning-for-microsoft-365"></a>Planejamento do Faça você mesmo para o Microsoft 365

Para planejar o Microsoft 365 por conta própria, confira as decisões de planejamento e design para estas áreas:

- Seu locatário do Microsoft 365

  Inclui o planejamento de suas conexões de rede com a Internet, suas identidades do Microsoft 365 e a integração com aplicativos, locais, Azure e outros elementos. Acesse [Assinaturas, licenças, contas e locatários](subscriptions-licenses-accounts-and-tenants-for-microsoft-cloud-offerings.md) para obter mais informações sobre ofertas de nuvem da Microsoft.

- Suporte para seus clientes

  Inclui autenticação baseada em certificado, gerenciamento de dispositivo móvel, opções de autenticação e colaboração entre locatários. Vá para [Autenticação baseada em certificado](microsoft-365-client-support-certificate-based-authentication.md) para obter mais informações.

- Suporte para autenticação moderna híbrida

  Inclui o planejamento da autenticação moderna ao usar configurações híbridas dos principais aplicativos de produtividade do Microsoft 365. Acesse a visão [geral da autenticação moderna híbrida e os pré-requisitos](hybrid-modern-auth-overview.md) para obter mais informações.

- Clientes e servidores mais antigos do Office

  Inclui informações de migração para produtos de cliente e servidor do Office 2007 e Office 2010. Vá para [Planejar a atualização](plan-upgrade-previous-versions-office.md) para obter mais informações.

Você também pode entrar em sua assinatura do Microsoft 365 e usar os guias de instalação do [Microsoft 365 e do Office 365.](setup-guides-for-microsoft-365.md)
