# <a name="plan-for-microsoft-365-enterprise"></a>Planejar Microsoft 365 Enterprise

Ao mover uma organização corporativa para Microsoft 365, é importante planejar e tomar as principais decisões de design que simplificarão a implantação de TI e a adoção do usuário. 

## <a name="planning-with-microsoft-365-fasttrack"></a>Planejando com Microsoft 365 FastTrack

[O FastTrack para Microsoft 365](https://www.microsoft.com/fasttrack/microsoft-365) é o melhor método para obter ajuda da Microsoft para planejar sua Microsoft 365 implantação. O FastTrack pode ajudá-lo nas considerações de design mais comuns e pode responder a perguntas ao longo do caminho. 

>[!Note]
>Você também pode obter ajuda de um [parceiro da Microsoft.](https://www.microsoft.com/solution-providers/home)
>

## <a name="do-it-yourself-planning-for-microsoft-365"></a>Planejamento do faça-você mesmo para Microsoft 365

Para planejar a Microsoft 365 por conta própria, passo a passo pelas decisões de planejamento e design dessas áreas:

- Seu locatário do Microsoft 365

  Inclui o planejamento de suas conexões de rede com a Internet, suas identidades Microsoft 365 e integração com aplicativos, local, Azure e outros elementos. Acesse [Assinaturas, licenças, contas e locatários](subscriptions-licenses-accounts-and-tenants-for-microsoft-cloud-offerings.md) para ofertas de nuvem da Microsoft para obter mais informações.

- Suporte para seus clientes

  Inclui autenticação baseada em certificado, gerenciamento de dispositivo móvel, opções de autenticação e colaboração entre locatários. Acesse [Autenticação baseada em certificado para](microsoft-365-client-support-certificate-based-authentication.md) obter mais informações.

- Suporte para autenticação moderna híbrida

  Inclui o planejamento da autenticação moderna ao usar configurações híbridas de aplicativos de Microsoft 365 de produtividade. Acesse [Visão geral de autenticação moderna híbrida e pré-requisitos](hybrid-modern-auth-overview.md) para obter mais informações.

- Servidores Office clientes e servidores mais antigos

  Inclui informações de migração para Office 2007 e Office produtos de cliente e servidor 2010. Vá para [Planejar sua atualização para](plan-upgrade-previous-versions-office.md) obter mais informações.

Você também pode entrar na sua assinatura Microsoft 365 e usar os guias de Instalação para serviços Microsoft 365 [e Office 365.](setup-guides-for-microsoft-365.md)
