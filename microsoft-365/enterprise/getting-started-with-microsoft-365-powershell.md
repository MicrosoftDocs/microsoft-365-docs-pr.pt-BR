# <a name="get-started-with-powershell-for-microsoft-365"></a>Introdução ao PowerShell para o Microsoft 365

*Este artigo se aplica tanto ao Microsoft 365 Enterprise quanto ao Office 365 Enterprise.*

Você pode usar comandos e scripts no PowerShell para a Microsoft 365 para gerenciar o Microsoft 365 e simplificar seu trabalho diário. Saiba por que o PowerShell para Microsoft 365 é crucial para gerenciar o Microsoft 365, como se conectar à sua assinatura do Microsoft 365 e criar relatórios e onde obter mais informações e ajuda da comunidade Microsoft 365.
  
Selecione estes tópicos:
  
- [**Por que você precisa usar o PowerShell para Microsoft 365**](why-you-need-to-use-microsoft-365-powershell.md)
    
    Inicie aqui se você for novo no PowerShell para o Microsoft 365. Saiba por que você deve usar o PowerShell para o Microsoft 365.
    
- [**Conecte-se ao Microsoft 365 com o PowerShell**](connect-to-microsoft-365-powershell.md)
    
    Comece aqui para se conectar à sua assinatura do Microsoft 365 usando o PowerShell para Microsoft 365 e tarefas administrativas da linha de comando.
    
- [**Conectar-se a todos os serviços do Microsoft 365 usando uma única janela do PowerShell**](connect-to-all-microsoft-365-services-in-a-single-windows-powershell-window.md)
    
    Você pode gerenciar o Microsoft 365 em janelas separadas para Skype for Business Online, SharePoint Online, Microsoft Exchange Online e Microsoft 365 contas e licenças. Ou você pode gerenciá-los em uma única janela. Este artigo explica como.
    
- [**Usar o PowerShell para criar relatórios no Microsoft 365**](use-windows-powershell-to-create-reports-in-microsoft-365.md)
    
    Inicie aqui se você tiver instalado os módulos do PowerShell para o Microsoft 365 e deseja saber mais sobre como usar comandos de automação para criar relatórios rapidamente.
    
- [**Referências de Cmdlet para serviços do Microsoft 365**](cmdlet-references-for-microsoft-365-services.md)
    
    Saiba mais sobre os cmdlets dos módulos do PowerShell para o Microsoft 365.
    
- [**Recursos da comunidade Microsoft 365 para PowerShell**](microsoft-365-powershell-community-resources.md)
    
    Inicie aqui para se conectar à comunidade do PowerShell e obter ajuda ou mais informações sobre como usar o PowerShell para o Microsoft 365.
