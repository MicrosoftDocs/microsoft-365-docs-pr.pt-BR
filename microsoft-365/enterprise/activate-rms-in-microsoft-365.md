# <a name="activate-rights-management-in-the-admin-center"></a>Ativar o gerenciamento de direitos no centro de administração

Você deve ativar o RMS (Serviço de Gerenciamento de Direitos) antes de poder usar os recursos de GERENCIAMENTO de Direitos de Informação (IRM) dos aplicativos e serviços do Microsoft 365. Depois de ativar o RMS, sua organização pode começar a proteger documentos e emails importantes usando o Azure RMS. Essa solução de proteção de informações pode proteger todos os tipos de arquivo e integra-se a aplicativos cliente, como o Excel, o Microsoft Word e outros, o Exchange Online e o SharePoint Online, além de servidores como o Microsoft Exchange e o Microsoft SharePoint.
  
> [!TIP]
> Se você não tiver certeza se precisa do Gerenciamento de Direitos, verifique se sua organização tem um ou mais desses problemas ou requisitos [de negócios.](https://docs.microsoft.com/azure/information-protection/what-is-azure-rms#business-problems-solved-by-azure-rights-management) 
  
Use estes links para obter mais informações sobre o RMS:
  
- Para saber mais sobre o RMS, consulte [O que é o Azure Rights Management.](https://docs.microsoft.com/rights-management/understand-explore/what-is-azure-rms)

- Se você for novo no RMS, consulte Visão [geral do Azure Rights Management](https://docs.microsoft.com/rights-management/understand-explore/azure-rights-management).

- Para uma visão geral das etapas de implantação, consulte o mapa de implantação do [Azure Rights Management.](https://docs.microsoft.com/rights-management/plan-design/deployment-roadmap)

- Para obter instruções sobre como ativar o RMS para o Microsoft 365, confira Ativar o serviço de proteção da Proteção de Informações do [Azure.](https://docs.microsoft.com/azure/information-protection/activate-service)
