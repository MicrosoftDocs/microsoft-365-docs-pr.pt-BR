# <a name="internet-sites-in-microsoft-azure-using-sharepoint-server-2013"></a>Sites da Internet no Microsoft Azure usando o SharePoint Server 2013

 Sites da Internet que usam o SharePoint Server 2013 se beneficiam por serem hospedados nos Serviços de Infraestrutura do Azure. Este artigo fornece recursos para projetar e implementar essa solução.

## <a name="using-azure-infrastructure-services-for-internet-sites"></a>Usando os Serviços de Infraestrutura do Azure para sites da Internet

O Microsoft Azure fornece uma opção atraente para hospedar sites da Internet com base no SharePoint Server 2013. As vantagens incluem o seguinte:

- Concentre-se em desenvolver um ótimo site em vez de criar infraestrutura.

- Flexibilidade para dimensionar sua solução com base na demanda, dimensionando e ampliando.

- Pagar somente pelos recursos necessários e usá-los.

- Aproveite o Azure Active Directory para contas de clientes.

- Adicione recursos que não estão disponíveis atualmente no Microsoft 365, como relatórios e análises profundas.
