# <a name="find-help-for-your-modern-desktop-deployment"></a>Encontre ajuda para sua implantação de área de trabalho moderna 

Se você estiver planejando sua implantação do Windows 10 com o Office 365 ProPlus, há várias opções para obter mais ajuda de parceiros certificados da Microsoft e do [Microsoft FastTrack](https://www.microsoft.com/fasttrack).

Veja abaixo um mapa dos serviços disponíveis alinhados ao processo de [Implantação de Área de Trabalho Moderna](http://www.aka.ms/howtoshift) para Windows 10, Office 365 ProPlus e Enterprise + Security com parceiro existente e ofertas do FastTrack. Em muitos casos, esses serviços estão incluídos em sua assinatura qualificada do Microsoft 365, Office 365 ou Windows 10 Enterprise. Para serviços liderados por parceiros, como a Avaliação de Área de Trabalho Moderna, como organização qualificada, a Microsoft fornece um comprovante para serviços prestados pelo parceiro. Os detalhes de cada oferta podem ser encontrados nos links abaixo.
