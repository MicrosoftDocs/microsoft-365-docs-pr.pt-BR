# <a name="find-help-for-your-windows-10-and-microsoft-365-apps-for-enterprise-deployment"></a>Obtenha ajuda para o Windows 10 e o Microsoft 365 Apps para implantação corporativa 

Se você estiver planejando sua implantação do Windows 10 com o Microsoft 365, há várias opções para obter mais ajuda de parceiros certificados da Microsoft e do [Microsoft FastTrack](https://www.microsoft.com/fasttrack).

Abaixo há um mapa dos serviços disponíveis alinhados com o processo de [Implantação da área de trabalho](https://www.aka.ms/howtoshift) do Windows 10, Microsoft 365 e Enterprise Mobility + Security com uma parceria existente e ofertas do FastTrack. Em muito casos, estes serviços estão inclusos na sua assinatura qualificada do Microsoft 365, Office 365 ou Windows 10 Enterprise. Para serviços liderados por parceiros, como o Windows 10 e o Microsoft 365 Apps para avaliação da área de trabalho corporativa, como uma organização qualificada, a Microsoft fornece um voucher para os serviços executados pelo parceiro. Detalhes sobre cada oferta podem ser encontrados através dos links abaixo.
