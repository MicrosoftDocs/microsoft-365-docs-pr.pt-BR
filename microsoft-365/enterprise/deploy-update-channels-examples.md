# <a name="deployment-and-update-channel-example-configurations"></a>Exemplo de configurações de canal de implantação e atualização

Escolher quais canais de atualização devem ser usados para o Windows 10 e os aplicativos do Microsoft 365 podem depender do tipo de organização e de onde no ciclo de desenvolvimento você deseja implantar e usar novos recursos e funcionalidades. Encontre os canais de pré-lançamento e de produção que melhor atendem às suas necessidades.

## <a name="pre-release-channels"></a>Canais de pré-lançamento

| Oferta de cliente/canal | Windows 10 | Aplicativos do Microsoft 365 para Empresas (Windows 10) |
|:-------|:-------|:-----|
| Perfeito para desenvolvedores e usuários avançados. <br><br> Seja o primeiro a acessar as versões mais recentes no ciclo inicial de desenvolvimento com o código mais novo. <br><br> Haverá falhas e algumas instabilidades. | Desenvolvimento | N/A |
| Certo para os pioneiros e profissionais de TI que desejam compilações mais confiáveis que ainda estejam em desenvolvimento. <br><br> Veja o que vem a seguir e ajude a validar novos recursos. | Canal beta | Canal beta |
| Ideal para as pessoas que desejam acesso antecipado a versões futuras. <br><br> Onde as empresas visualizam e validam versões futuras antes da implantação abrangente. <br><br> Estes têm suporte. <br>  | Versão prévia | Canal Atual (Visualização) <br><br> Canal Empresarial Semestral (Visualização)|
||||

## <a name="production-channels-for-broad-deployment"></a>Canais de produção para implantação ampla

Clique no link no **Exemplo** coluna para percorrer os estágios e os grupos de implantação de um exemplo de organização.

| Oferta de cliente/canal | Windows 10 | Aplicativos do Microsoft 365 para Empresas (Windows 10) | Exemplo |
|:-------|:-------|:-----|:-------|
| Ideal para as pessoas que desejam acesso antecipado assim que tiverem prontas. | Canal Semestral | [Canal Atual](https://docs.microsoft.com/deployoffice/overview-update-channels#current-channel-overview) | [Versões mais recentes](deploy-update-channels-examples-rapid-deploy.md) |
| Ideal para empresas que desejam acesso antecipado com mais previsibilidade. | Canal Semestral | [Canal Empresarial Mensal](https://docs.microsoft.com/deployoffice/overview-update-channels#monthly-enterprise-channel-overview) |  |
| Ideal para empresas que precisam de grandes testes de TI antes de cada atualização. | Canal Semestral | [Canal Empresarial Semestral](https://docs.microsoft.com/deployoffice/overview-update-channels#semi-annual-enterprise-channel-overview) |  |
|||||
