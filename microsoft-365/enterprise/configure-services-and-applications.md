# <a name="configure-microsoft-365-enterprise-services-and-applications"></a>Configurar aplicativos e serviços do Microsoft 365 Enterprise

Nossas [instruções básicas de configuração](https://docs.microsoft.com/microsoft-365/admin/setup/setup) ajudam você a fazer com que todos usem os serviços e aplicativos do Microsoft 365 no menor tempo possível. Às vezes, é preferível configurar as coisas antes que todos comece a usá-las. Por exemplo, se você quiser configurar roteamento de email, armazenamento de arquivos ou políticas de compartilhamento. 
  
Se você quiser ajuda para configurar o Microsoft 365, use o **[FastTrack](https://www.microsoft.com/fasttrack/microsoft-365)** ou os guias de instalação do [Microsoft 365 e do Office 365.](setup-guides-for-microsoft-365.md)
