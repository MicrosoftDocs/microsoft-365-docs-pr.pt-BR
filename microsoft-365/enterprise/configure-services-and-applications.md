# <a name="configure-microsoft-365-enterprise-services-and-applications"></a>Configurar Microsoft 365 Enterprise serviços e aplicativos

Nossas [instruções básicas de configuração](../admin/setup/setup.md) ajudam a fazer com que todos usem seus serviços Microsoft 365 e aplicativos no menor tempo possível. Às vezes, é preferível configurar as coisas antes de todos começarem a usá-las. Por exemplo, se você quiser configurar o roteamento de email, armazenamento de arquivos ou políticas de compartilhamento. 
  
Se você quiser ajuda para Microsoft 365 configuração, use **[o FastTrack](https://www.microsoft.com/fasttrack/microsoft-365)** ou os guias de Instalação para serviços Microsoft 365 [e Office 365 .](setup-guides-for-microsoft-365.md)
