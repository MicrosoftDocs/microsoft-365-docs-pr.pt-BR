# <a name="manage-microsoft-365-with-powershell"></a>Gerenciar o Microsoft 365 com o PowerShell

*Este artigo se aplica tanto ao Microsoft 365 Enterprise quanto ao Office 365 Enterprise.*

O PowerShell para o Microsoft 365 é uma ferramenta de gerenciamento poderosa que complementa o centro de administração do Microsoft 365. Por exemplo, você pode usar a automação do PowerShell para gerenciar facilmente várias contas de usuário e licenças e criar relatórios.

Selecione entre os tópicos a seguir para saber como usar o PowerShell para gerenciar o Microsoft 365:
  
- [**Introdução**](getting-started-with-microsoft-365-powershell.md)

    Comece aqui se você não estiver familiarizado com o PowerShell para o Microsoft 365 e quiser instalar os módulos do Microsoft 365 e se conectar à sua assinatura.

- [**Contas de usuário, licenças e grupos**](manage-user-accounts-and-licenses-with-microsoft-365-powershell.md)

    Comece aqui se quiser saber mais sobre como usar comandos de automação para gerenciar contas de usuário, licenças e grupos.

- [**SharePoint**](manage-sharepoint-online-with-microsoft-365-powershell.md)

    Comece aqui se quiser usar comandos de automação para gerenciar o SharePoint.

- [**Exchange Online**](/powershell/exchange/exchange-online-powershell)

    Comece aqui se quiser gerenciar o Exchange Online.

- [**Migração de email**](use-powershell-for-email-migration-to-microsoft-365.md)

    Comece aqui se quiser migrar seu email de sistemas pré-existentes.

- [**Centro de segurança e conformidade**](/powershell/exchange/scc-powershell)

    Comece aqui se quiser gerenciar os recursos do Centro de Conformidade & segurança.

- [**Parceiros de Permissões de Acesso Delegado (DAP)**](manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-dap-p.md)

    Comece aqui se quiser usar parceiros CSP (Provedor de Soluções de Nuvem e Syndication) para gerenciar seus locatários de clientes do Microsoft 365.

- [**Skype for Business Online**](manage-skype-for-business-online-with-microsoft-365-powershell.md)

    Comece aqui para gerenciar o Skype for Business Online.
