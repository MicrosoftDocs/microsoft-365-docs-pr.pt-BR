# <a name="manage-microsoft-365-with-powershell"></a>Gerenciar o Microsoft 365 com o PowerShell

*Este artigo se aplica tanto ao Microsoft 365 Enterprise quanto ao Office 365 Enterprise.*

O PowerShell para Microsoft 365 é uma poderosa ferramenta de gerenciamento que complementa o centro de administração do Microsoft 365. Por exemplo, você pode usar a automação do PowerShell para gerenciar facilmente várias contas de usuário e licenças e para criar relatórios.

Selecione entre os tópicos a seguir para saber como usar o PowerShell para gerenciar o Microsoft 365:
  
- [**Introdução**](getting-started-with-microsoft-365-powershell.md)

    Inicie aqui se você não estiver familiarizado com o PowerShell para o Microsoft 365 e deseja instalar os módulos do Microsoft 365 e se conectar à sua assinatura.

- [**Contas de usuário, licenças e grupos**](manage-user-accounts-and-licenses-with-microsoft-365-powershell.md)

    Inicie aqui se quiser saber mais sobre como usar comandos de automação para gerenciar contas de usuário, licenças e grupos.

- [**Microsoft Office SharePoint Online**](manage-sharepoint-online-with-microsoft-365-powershell.md)

    Inicie aqui se você deseja usar comandos de automação para gerenciar o SharePoint.

- [**Exchange Online**](https://docs.microsoft.com/powershell/exchange/exchange-online-powershell)

    Inicie aqui se você quiser gerenciar o Exchange Online.

- [**Migração de email**](use-powershell-for-email-migration-to-microsoft-365.md)

    Inicie aqui se você quiser migrar seus emails de sistemas preexistentes.

- [**Centro de segurança e conformidade**](https://docs.microsoft.com/powershell/exchange/scc-powershell)

    Inicie aqui se você quiser gerenciar os recursos de segurança & centro de conformidade.

- [**Parceiros com permissões de acesso delegado (DAP)**](manage-microsoft-365-with-windows-powershell-for-delegated-access-permissions-dap-p.md)

    Inicie aqui se você deseja usar os parceiros de distribuição e provedor de soluções de nuvem (CSP) para gerenciar seus locatários do cliente Microsoft 365.

- [**Skype for Business Online**](manage-skype-for-business-online-with-microsoft-365-powershell.md)

    Comece aqui para gerenciar o Skype for Business online.
