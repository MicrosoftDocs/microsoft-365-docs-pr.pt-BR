<span data-ttu-id="eb465-107">O Azure AD Connect é a ferramenta da Microsoft com suporte que guia você pela sincronização somente das identidades que você realmente precisa de ambientes AD DS de floresta única ou de várias florestas para o seu locatário do Azure AD.</span><span class="sxs-lookup"><span data-stu-id="eb465-107">Azure AD Connect is the supported Microsoft tool that guides you through synchronizing only the identities you really need from single or multi-forest AD DS environments to your Azure AD tenant.</span></span> <span data-ttu-id="eb465-108">A figura a seguir mostra o processo básico para a sincronização do Azure AD Connect.</span><span class="sxs-lookup"><span data-stu-id="eb465-108">The following figure shows the basic process for Azure AD Connect synchronization.</span></span>

The following options are options:</span></span>

- <span data-ttu-id="eb465-p103">Com a **autenticação gerenciada**, o Azure AD lida com o processo de autenticação do login do usuário.     - <span data-ttu-id="eb465-120">A **Autenticação de passagem (PTA)** fornece uma solução de validação de senha simples para os serviços baseados no Azure AD.</span><span class="sxs-lookup"><span data-stu-id="eb465-120">**Pass-through Authentication (PTA)** provides a simple password validation solution for Azure AD-based services.</span></span> <span data-ttu-id="eb465-121">PTA usa um agente em execução em um ou mais servidores locais para validar as autenticações de usuário diretamente com seu AD DS local.</span><span class="sxs-lookup"><span data-stu-id="eb465-121">PTA uses an agent running on one or more on-premises servers to validate the user authentications directly with your on-premises AD DS.</span></span> <span data-ttu-id="eb465-122">Para saber mais, confira [Entrada do usuário com autenticação de passagem do Azure Active Directory](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-pass-through-authentication).</span><span class="sxs-lookup"><span data-stu-id="eb465-122">For more information, see [User sign-in with Azure Active Directory Pass-through Authentication](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-pass-through-authentication).</span></span>
- <span data-ttu-id="eb465-p106">Com a **autenticação federada**, o processo de autenticação é redirecionado para outro provedor de identidade por meio de um servidor de federação de identidade, como os Serviços de Federação do Active Directory (AD FS) para o login de um usuário.
