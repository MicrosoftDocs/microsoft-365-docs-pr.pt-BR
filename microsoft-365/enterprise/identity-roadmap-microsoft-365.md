# <a name="identity-roadmap-for-microsoft-365"></a>Mapa da identidade do Microsoft 365

No Microsoft 365 para empresas, uma infraestrutura de identidade bem planejada e executada abre o caminho para uma segurança mais forte, incluindo a restrição do acesso às cargas de trabalho de produtividade e seus dados apenas a usuários e dispositivos autenticados.

Para uma visão geral dos modelos de identidade e autenticação do Microsoft 365 para empresas, assista a este vídeo.

<p> </p>

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE2Pjwu]

Para obter informações sobre os recursos de identidade de cada plano empresarial do Microsoft 365, a função do Azure Active Directory, os componentes locais e baseados na nuvem e as configurações de autenticação mais comuns, consulte o cartaz infraestrutura de identidade [.](../downloads/m365e-identity-infra.pdf)

[![Pôster sobre a Infraestrutura de Identidade](../downloads/m365e-identity-infra.png)](../downloads/m365e-identity-infra.pdf)

Revise este cartaz de duas páginas para aumentar rapidamente os conceitos e configurações de identidade do Microsoft 365 para empresas.

Você também pode [baixar esse cartaz e](https://github.com/MicrosoftDocs/microsoft-365-docs/raw/public/microsoft-365/downloads/m365e-identity-infra.pdf) imprimi-lo em formato de carta, legal ou tablóide (11 x 17).
