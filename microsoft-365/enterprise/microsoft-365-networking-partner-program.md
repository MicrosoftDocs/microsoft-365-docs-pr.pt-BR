# <a name="microsoft-365-networking-partner-program"></a>Programa de Parceiros de Redes do Microsoft 365

A conectividade de rede tem um impacto direto na capacidade dos usuários de trabalhar rapidamente, colaborar de forma eficaz e simplificar os processos de negócios com o Microsoft 365. Para clientes em qualquer estágio de sua transformação digital, o design de rede é um aspecto crítico que deve ser abordado proativamente antes de problemas afetarem negativamente a experiência do usuário.

À medida que os clientes adotam o Microsoft 365 para a produtividade dos negócios, a Microsoft observa uma tendência comum de que o desempenho da rede e a experiência de colaboração do usuário final resultante são diretamente influenciados pelas soluções de rede no caminho entre o usuário e o Microsoft 365. Para ajudar os parceiros a projetar soluções de rede ideais e ajudar os clientes a tomar decisões informadas sobre essas soluções, construímos o Programa de Parceiros de Rede do Microsoft 365.

O Programa de Parceiros de Rede do Microsoft 365 aprofunda nossa colaboração com parceiros de rede e identifica os principais produtos e soluções que seguem os requisitos de rede, recomendações e práticas recomendadas do Microsoft 365. O objetivo do programa parceiro de rede do Microsoft 365 é facilitar a capacidade do cliente de melhorar sua experiência do Microsoft 365 por meio da fácil descoberta de soluções de parceiros validadas que demonstram consistentemente o alinhamento aos princípios-chave para a conectividade ideal do Microsoft 365 nas implantações do cliente.

Para modernizar redes corporativas para uma ótima conectividade com o Microsoft 365, os clientes geralmente dependem de provedores de soluções de rede, serviços de segurança locais ou baseados em nuvem e integradores de sistema para planejar, projetar e implementar a conectividade de rede para serviços de nuvem. Os clientes geralmente questionam a Microsoft se sua arquitetura de rede e soluções funcionam com o Microsoft 365 e se eles se alinham aos Princípios de Conectividade de Rede da Microsoft para [o Microsoft 365](./microsoft-365-network-connectivity-principles.md).

O Programa de Parceiros de Rede do Microsoft 365 ajuda a responder a essas perguntas e demonstra o compromisso da Microsoft em ajudar nossos clientes a criar e otimizar sua arquitetura de rede para a melhor experiência do Microsoft 365. A equipe do Microsoft 365 está trabalhando com muitos parceiros do setor de rede para ajudar a garantir que os princípios-chave para a conectividade ideal sejam integrados de forma nativa em seus produtos e soluções de rede.

O Programa de Parceiros de Rede do Microsoft 365 designa soluções de rede qualificadas como "Funciona com o Microsoft 365". Somente dispositivos ou soluções que atendem aos rigorosos requisitos de teste da Microsoft, têm experiências perfeitas com a configuração e demonstram uma experiência de rede de alta qualidade quando usados com o Microsoft 365 receberão a designação "Works with Microsoft 365". Essas soluções de parceiros validadas estão listadas na página Programa de Parceiros de Rede do [Microsoft 365](https://www.microsoft.com/microsoft-365/partners/O365networkingpartners), juntamente com detalhes da solução e links para saber mais.

Se você for um cliente do Microsoft 365, poderá tornar seu provedor de soluções de rede ciente desse programa por meio desta página em [https://aka.ms/Office365NPP]() .

Para provedores de soluções de rede que querem saber mais sobre o Programa de Parceiros de Rede do Microsoft 365, preencha o formulário em [https://aka.ms/NPPsignup](https://aka.ms/NPPsignup) . Obrigado por seu interesse e estamos ansiosos por uma bela parceria.
