# <a name="step-4-simplify-password-resets"></a>Etapa 4: Simplificar redefinições de senha

*Esta etapa é opcional e se aplica às versões E3 e E5 do Microsoft 365 Enterprise*

![](./media/deploy-foundation-infrastructure/identity_icon-small.png)

Nesta etapa, você habilitará a redefinição de senha de autoatendimento (SSPR) para permitir que os usuários redefinam ou desbloqueiem senhas ou contas. Para alertá-lo sobre uso incorreto ou abuso, é possível usar relatórios detalhados para monitorar quando os usuários acessam o sistema, junto com as notificações.

Confira as [instruções para habilitar a redefinição de senha](https://docs.microsoft.com/azure/active-directory/active-directory-passwords-best-practices).
