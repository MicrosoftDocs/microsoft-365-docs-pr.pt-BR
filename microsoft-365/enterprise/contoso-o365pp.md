## <a name="managing-the-deployment-and-updates"></a><span data-ttu-id="145a4-119">Gerenciar a implantação e as atualizações</span><span class="sxs-lookup"><span data-stu-id="145a4-119">Managing the deployment and updates</span></span>

<span data-ttu-id="145a4-p103">O Microsoft 365 Apps for enterprise tem um novo modelo de lançamento: o Office como serviço. To minimize any compatibility issues and to ensure their computers stayed up to date, Contoso deployed Windows and Office in two stages:</span></span> 

- <span data-ttu-id="145a4-p104">Para a primeira etapa, eles implantaram o Microsoft 365 Apps for enterprise para um pequeno conjunto de dispositivos representativos em toda a organização. - <span data-ttu-id="145a4-125">Quatro meses depois disso, após resolver todos os problemas graves com aplicativos, suplementos e hardware no grupo piloto, a Contoso implantou o Microsoft 365 Apps for enterprise no restante dos dispositivos da organização (o grupo amplo).</span><span class="sxs-lookup"><span data-stu-id="145a4-125">Four months later, after addressing all critical issues with apps, add-ins, and hardware in the pilot group, Contoso deployed Microsoft 365 Apps for enterprise to the rest of the devices in the organization (the broad group).</span></span> 

<span data-ttu-id="145a4-p105">Em vez de gerenciar as atualizações do Office com o Gerenciador de Configurações, a Contoso habilitou as atualizações automáticas da nuvem. <span data-ttu-id="145a4-p106">A Contoso seguiu a mesma abordagem de duas etapas para as atualizações de recursos que eles usavam para implantar o Office: os dispositivos no grupo piloto recebiam as atualizações de recursos quatro meses antes do que os dispositivos no restante da organização (o grupo amplo). To enable this for Office, Contoso used two recommended [update channels](https://docs.microsoft.com/DeployOffice/overview-update-channels):</span></span> 

- <span data-ttu-id="145a4-130">Canal Empresarial Semestral (Visualização) para atualizações do grupo piloto</span><span class="sxs-lookup"><span data-stu-id="145a4-130">Semi-Annual Enterprise Channel (Preview) for updates to the pilot group</span></span> 
- <span data-ttu-id="145a4-131">Canal Empresarial Semestral para atualizações para o grupo amplo.</span><span class="sxs-lookup"><span data-stu-id="145a4-131">Semi-Annual Enterprise Channel for updates to the broad group.</span></span> 

<span data-ttu-id="145a4-132">Como o Canal Empresarial Semestral (Visualização) lança uma versão do Microsoft 365 Apps para Grandes Empresas quatro meses antes do Canal Empresarial Semestral, a Contoso tem tempo para validar as atualizações sem precisar gerenciá-las.</span><span class="sxs-lookup"><span data-stu-id="145a4-132">Because the Semi-Annual Enterprise Channel (Preview) releases a version of Microsoft 365 Apps for enterprise four months earlier than the Semi-Annual Enterprise Channel, Contoso has time to validate the updates without having to manage them.</span></span> 

## <a name="deployment-process"></a><span data-ttu-id="145a4-133">Processo de implantação</span><span class="sxs-lookup"><span data-stu-id="145a4-133">Deployment process</span></span>

6. <span data-ttu-id="145a4-143">Eles usaram o recurso interno no pacote do Office para remover automaticamente todas as versões existentes do MSI do Office antes de instalar o Microsoft 365 Apps for enterprise.</span><span class="sxs-lookup"><span data-stu-id="145a4-143">They used the built-in feature in the Office package to automatically remove all existing MSI versions of Office before installing Microsoft 365 Apps for enterprise.</span></span>
7. <span data-ttu-id="145a4-144">No Gerenciador de Configurações, eles implantaram os pacotes do Windows e do Office em pontos de distribuição em toda a rede e executaram as sequências de tarefas de implantação do Gerenciador de Configurações para implantar o pacote piloto do Microsoft 365 Apps for enterprise no grupo piloto.</span><span class="sxs-lookup"><span data-stu-id="145a4-144">In Configuration Manager, they deployed the Windows and Office packages to distribution points across their network, and then ran the Configuration Manager deployment task sequences to deploy the pilot Microsoft 365 Apps for enterprise package to the pilot group.</span></span>
8. <span data-ttu-id="145a4-145">Depois de abordar problemas de compatibilidade com o grupo piloto, a Contoso executou as sequências de tarefas para implantar o pacote amplo do Microsoft 365 Apps for enterprise no grupo amplo.</span><span class="sxs-lookup"><span data-stu-id="145a4-145">After addressing any compatibility issues with the pilot group, Contoso ran the task sequences to deploy the broad Microsoft 365 Apps for enterprise package to the broad group.</span></span>
