# <a name="microsoft-365-endpoints"></a>Pontos de extremidade do Microsoft 365

*Esse artigo se aplica ao Microsoft 365 Enterprise e ao Office 365 Enterprise.*

Pontos de extremidade são o conjunto de endereços IP de destino, nomes de domínio DNS e URLs para o tráfego do Microsoft 365 na Internet. 

Para otimizar o desempenho dos serviços baseados em nuvem do Microsoft 365, esses pontos de extremidade precisam de um tratamento especial pelos navegadores clientes e pelos dispositivos em sua rede de borda. Esses dispositivos incluem firewalls, dispositivos de quebra e inspeção de SSL e inspeção de pacotes e sistemas de prevenção contra perda de dados.

Consulte Gerenciando pontos de extremidade do [Microsoft 365](managing-office-365-endpoints.md) para obter detalhes.

Atualmente, há cinco nuvens diferentes do Microsoft 365. Esta tabela leva você à lista de pontos de extremidade de cada um deles.

| Nuvem | Descrição |
|:-------|:-----|
| [Pontos de extremidade mundiais](urls-and-ip-address-ranges.md) | Os pontos de extremidade das assinaturas do Microsoft 365 em todo o mundo, que incluem o United States Government Community Cloud (GCC). |
| [Pontos de extremidade do US Government DoD](microsoft-365-u-s-government-dod-endpoints.md) | Os pontos de extremidade das assinaturas do Departamento de Defesa dos Estados Unidos (DoD). |
| [Pontos de extremidade do US Government GCC High](microsoft-365-u-s-government-gcc-high-endpoints.md) | Os pontos de extremidade das assinaturas da United States Government Community Cloud High (GCC alto). |
| [Pontos de extremidade do Microsoft 365 operados pela 21Vianet](urls-and-ip-address-ranges-21vianet.md) | Os pontos de extremidade do Microsoft 365 operados pela 21Vianet, projetados para atender às necessidades do Microsoft 365 na China. |
| [Pontos de extremidade do Microsoft 365 Germany](microsoft-365-germany-endpoints.md) | Os pontos de extremidade uma nuvem separada na Europa para a maioria dos clientes regulamentados na Alemanha, União Europeia (UE) e Associação Europeia de Livre Comércio (EFTA). |
|||

Para obter a lista mais recente de pontos de extremidade para a nuvem do Microsoft 365, confira o endereço IP do [Office 365](microsoft-365-ip-web-service.md)e a URL do serviço Web.

Para pontos de extremidade adicionais, confira estes artigos:

- [Pontos de extremidade adicionais não incluídos nos serviços Web](additional-office365-ip-addresses-and-urls.md)
- [Solicitações de rede do Office 2016 para Mac](network-requests-in-office-2016-for-mac.md)

Se você for um fornecedor de equipamentos de rede, participe do Programa de Parceiros de Rede do [Office 365.](microsoft-365-networking-partner-program.md) Inscreva-se no programa para criar princípios de conectividade de rede do Microsoft 365 em seus produtos e soluções.
