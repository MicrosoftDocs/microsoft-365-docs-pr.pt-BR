# <a name="get-access-to-microsoft-bookings"></a>Obter acesso ao Microsoft Bookings

Este tópico descreve como obter acesso ao Bookings pela primeira vez. Ele também mostra como ativar ou desativar o Bookings em sua organização. Ele também explica como você pode atribuir licenças aos usuários para que eles possam acessar e usar o Bookings.

## <a name="access-bookings-for-the-first-time"></a>Acessar o Bookings pela primeira vez

Se o administrador de TI tiver concedido acesso ao Bookings, você poderá acessar o aplicativo por meio do Office online.

Na primeira vez que usar [o Bookings,](https://outlook.office.com/bookings/onboarding) você criará um calendário e configurará sua empresa.

1. Selecione **Obter agora e** escolha, Adicionar um calendário de **reserva.** Se essa não for a primeira vez nesta página, você poderá selecionar outro calendário de reserva ou pesquisar outro calendário de reserva.

:::image type="content" source="media/bookings - get it now.png" alt-text="Screenshot: Get it now bookings page. Selecione Obter agora para começar a usar o Bookings":::

2. Insira seu nome de negócios e tipo de negócios e selecione **Continuar**.

Agora você está pronto para configurar o Bookings para sua organização. Você pode chegar à página do Bookings com este [link,](https://outlook.office.com/bookings/onboarding)no iniciador de aplicativos ou no office.com. Use os tópicos a seguir para continuar configurando seus recursos do Bookings.

- [Agendar fechamentos comerciais, folgas e períodos de férias](schedule-closures-time-off-vacation.md)
- [Configurar as políticas de agendamento](set-scheduling-policies.md)
- [Definir o horário de trabalho dos funcionários](employee-hours.md)
