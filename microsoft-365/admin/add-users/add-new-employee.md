# <a name="add-a-new-employee-to-microsoft-365"></a>Adicionar um novo funcionário ao Microsoft 365

Este artigo ajuda você a integrar um novo funcionário ao Microsoft 365 para empresas. Assumimos que você é um Administrador e já concluiu a configuração do [Microsoft 365](../setup/setup.md)e agora você tem alguém novo in joining sua empresa.
  
Você está no lugar certo se seu novo funcionário precisa do Microsoft 365 e está usando um plano do [Microsoft 365](https://products.office.com/business/compare-office-365-for-business-plans) que permite instalar aplicativos do Office como o Word e o Excel em um computador. 
  
 **Não é um administrador?** [Saiba mais sobre o Microsoft 365 e](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) ajuda os usuários comerciais e a sua empresa com a configuração. 
  
 **Nenhum aplicativo do Office em seu plano?** Siga as etapas abaixo, mas ignore as seções para instalar aplicativos. Em vez disso, use [as versões Online do Office.](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) 
  
Aqui está uma visão geral rápida: 
  
|**Etapa**|**Por que fazer isso?**|
|:-----|:-----|
|[Etapa 1: Criar uma conta do Microsoft 365 para o funcionário](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Sempre que um novo funcionário ingressar em sua empresa, crie uma conta para ele para que ele possa começar a usar o Microsoft 365.  <br/> |
|[Etapa 2: dar ao funcionário sua ID de usuário e senha](#step-2-give-the-employee-their-user-id-and-password) <br/> |Ao criar uma conta, você obterá uma ID e uma senha que você pode passar ao funcionário para que ele possa entrar.  <br/> |
|[Etapa 3: Explicar onde entrar](#step-3-explain-where-to-sign-in) <br/> |O local de login é [https://www.office.com](https://www.office.com) <br/> |
|[Etapa 4: Ajude seu funcionário a começar](#step-4-help-your-employee-get-started) <br/> |Deixe seu funcionário saber como usar o OneDrive ou qualquer site de equipe em sua organização.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Etapa 1: Criar uma conta do Microsoft 365 para o funcionário


Para obter instruções, [consulte Adicionar usuários e atribuir licenças ao mesmo tempo.](add-users.md) Ao configurar seu novo funcionário, você pode optar por enviar detalhes de login para a conta pessoal do funcionário. Dessa forma, eles receberão um email da Equipe de Serviços Online da Microsoft que lhes diz como entrar no Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Etapa 2: dar ao funcionário sua ID de usuário e senha


A menos que você o tenha enviado para o endereço de email pessoal, imprima o nome de login e a senha do funcionário e entregue-o a ele. Ou diga a eles as informações por telefone.
  
Como eles ainda não terão acesso ao email, não envie as informações para esse endereço de email.
  
## <a name="step-3-explain-where-to-sign-in"></a>Etapa 3: Explicar onde entrar 


Assim como o Facebook, a Amazon ou o Gmail, seu funcionário se insinte para usar o Microsoft 365. Dê a eles as seguintes informações de login:
  
- Entre em [https://www.office.com](https://www.office.com) .
    
- Selecione **Entrar e** digite a ID de usuário e a senha.
    
## <a name="step-4-help-your-employee-get-started"></a>Etapa 4: Ajude seu funcionário a começar


Compartilhe com eles a configuração rápida funcionário do [Microsoft 365](https://support.microsoft.com/office/b9700090-ce64-4046-ab92-ce8488a7bc0f) para entrar, instalar software, configurar email e muito mais.
