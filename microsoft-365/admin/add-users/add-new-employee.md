# <a name="add-a-new-employee-to-microsoft-365"></a>Adicionar um novo funcionário ao Microsoft 365

Este artigo ajuda você a integrar um novo funcionário para Microsoft 365 para empresas. Presumimos que você seja um Administrador e já concluiu a Microsoft 365 [configuração](../setup/setup.md)e agora tem alguém novo para ingressar em sua empresa.
  
Você [está](https://products.office.com/business/compare-office-365-for-business-plans) no lugar certo se seu novo funcionário precisar de Microsoft 365 e estiver usando um plano Microsoft 365 que permite instalar aplicativos Office como o Word e o Excel em um computador. 
  
 **Não é um administrador?** [Aprenda a Microsoft 365 ajuda](https://support.microsoft.com/office/396b8d9e-e118-42d0-8a0d-87d1f2f055fb) os usuários de negócios e de residência com a configuração. 
  
 **Nenhum Office aplicativos em seu plano?** Siga as etapas abaixo, mas ignore as seções para instalar aplicativos. Use as [versões online do Office](https://support.microsoft.com/office/91a4ec74-67fe-4a84-a268-f6bdf3da1804) em vez disso. 
  
Aqui está uma visão geral rápida: 
  
|**Etapa**|**Por que fazer isso?**|
|:-----|:-----|
|[Etapa 1: criar uma Microsoft 365 para o funcionário](#step-1-create-a-microsoft-365-account-for-the-employee) <br/> |Sempre que um novo funcionário ingressar na sua empresa, crie uma conta para eles para que eles possam começar a usar Microsoft 365.  <br/> |
|[Etapa 2: dar ao funcionário sua ID de usuário e senha](#step-2-give-the-employee-their-user-id-and-password) <br/> |Ao criar uma conta, você obterá uma ID e uma senha que você pode passar para seu funcionário para que eles possam entrar.  <br/> |
|[Etapa 3: Explicar onde entrar](#step-3-explain-where-to-sign-in) <br/> |O local de login é [https://www.office.com](https://www.office.com) <br/> |
|[Etapa 4: Ajudar seu funcionário a começar](#step-4-help-your-employee-get-started) <br/> |Deixe seu funcionário saber como usar OneDrive ou qualquer site de equipe em sua organização.  <br/> |
   
## <a name="step-1-create-a-microsoft-365-account-for-the-employee"></a>Etapa 1: criar uma Microsoft 365 para o funcionário


Para obter instruções, [consulte Adicionar usuários e atribuir licenças ao mesmo tempo](add-users.md). Ao configurar seu novo funcionário, você pode optar por enviar detalhes de login para a conta pessoal do funcionário. Dessa forma, eles receberão um email da Equipe de Serviço do Microsoft Online que informa como fazer logoff no Microsoft 365.
  
## <a name="step-2-give-the-employee-their-user-id-and-password"></a>Etapa 2: dar ao funcionário sua ID de usuário e senha


A menos que você o tenha enviado para o endereço de email pessoal, imprima o nome de login e a senha do funcionário e entregue-o a eles. Ou diga a eles as informações por telefone.
  
Como eles ainda não terão acesso ao email, não envie as informações para esse endereço de email.
  
## <a name="step-3-explain-where-to-sign-in"></a>Etapa 3: Explicar onde entrar 


Assim como Facebook, Amazon ou Gmail, seu funcionário se insinuou para usar Microsoft 365. Dê a eles as seguintes informações de login:
  
- Entre em [https://www.office.com](https://www.office.com) .
    
- Selecione **Entrar** e digite a ID do usuário e a senha.
