# <a name="compare-ways-to-block-access"></a>Comparar maneiras de bloquear o acesso

Quando um funcionário sai da sua organização, em termos bons ou ruins, você precisa bloquear o acesso a Microsoft 365. Aqui estão algumas maneiras de fazer isso.
  
||||
|:-----|:-----|:-----|
|**Maneira de bloquear o acesso** <br/> |**Definição** <br/> |**Práticas recomendadas** <br/> |
|Bloquear a login  <br/> |Uma maneira de impedir que um usuário acesse Microsoft 365 é alterar o status de entrada para Entrar **bloqueado**. Isso impede que eles se inscrevem Microsoft 365 seus computadores e dispositivos móveis, embora ainda possam exibir emails e documentos baixados ou sincronizados anteriormente. Se você estiver usando o Blackberry Enterprise Serviço, também poderá desabilitar o acesso deles.  <br/> |Use quando um funcionário planeja sair da organização ou planeja tirar uma licença de longo prazo.  <br/> |
|Redefinir senha do usuário  <br/> |Outra maneira de impedir que um usuário acesse Microsoft 365 é redefinir sua senha. Isso impede que eles usem sua conta, embora ainda possam exibir emails e documentos baixados ou sincronizados anteriormente. Em seguida, você pode entrar como eles e alterar a senha para uma de suas escolhas.  <br/> |Use quando um funcionário sair de forma repentina e permanente e você sentir que há preocupação com dados comerciais.  <br/> |
|Remover todas as licenças atribuídas  <br/> |Outra opção é remover quaisquer licenças Microsoft 365 atribuídas ao usuário. Isso os impede de usar aplicativos e serviços como o pacote Office, Office aplicativos para a Web, Yammer e SharePoint Online. Eles ainda podem entrar, mas não podem usar esses serviços.  <br/> |Use quando você acha que esse usuário não precisa mais de acesso a recursos específicos Microsoft 365.  <br/> <br> **Importante:** Quando você remover uma licença, a caixa de correio do usuário será excluída em 30 dias.
