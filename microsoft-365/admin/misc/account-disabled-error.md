# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Obter um erro de conta desabilitada no Outlook na Web

Se você receber  o erro Sua conta foi desabilitada quando você tenta abrir o Outlook na Web (anteriormente conhecido como Outlook Web App), o administrador pode ter desabilitado seu acesso ao `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` Outlook na Web.

O administrador pode corrigir esse erro seguindo as etapas do tópico [Habilitar](https://docs.microsoft.com/exchange/recipients-in-exchange-online/manage-user-mailboxes/enable-or-disable-outlook-web-app)ou desabilitar o Outlook Web App para uma caixa de correio.
