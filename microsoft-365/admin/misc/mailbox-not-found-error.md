# <a name="getting-a-mailbox-not-found-error-in-outlook-on-the-web"></a>Obter uma caixa de correio erro não encontrado no Outlook na Web?

Se você estiver usando o Outlook na Web e receber uma Caixa de Correio por erro,  **a**  conta que você usou para se conectar ao Outlook na Web não tem uma licença do Exchange Online e, portanto, nenhuma caixa de correio está associada à conta. O administrador pode atribuir uma licença à sua conta seguindo estas etapas:

1. Abra o [centro de administração do Microsoft 365,](https://portal.office.com/adminportal/home#/homepage) vá para Usuários **ativos** na seção Usuários e selecione o usuário que está vendo o erro. 
2. Na página do usuário que é aberta, vá para  a seção Licenças e **Aplicativos,** selecione o valor de Local apropriado e atribua uma licença que contenha o Exchange Online (expanda a licença para ver seus detalhes). Quando terminar, clique em **Salvar alterações.**
