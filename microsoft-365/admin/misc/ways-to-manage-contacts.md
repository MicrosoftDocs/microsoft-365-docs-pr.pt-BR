# <a name="quick-help-ways-to-manage-contacts"></a>Ajuda rápida: maneiras de gerenciar contatos

Há várias maneiras de trabalhar com contatos no Microsoft 365, dependendo em grande parte de como você usa os contatos. Aqui está uma comparação das diferentes maneiras de ajudá-lo a escolher a opção certa para sua organização.
  
|||||
|:-----|:-----|:-----|:-----|
|**Recurso** <br/> |**Descrição** <br/> |**Vantagens** <br/> |**Desvantagens** <br/> |
|**Contatos do Microsoft 365** <br/> |Use os contatos do Microsoft 365 para armazenar seus clientes ou contatos comerciais e eles aparecerão em Pessoas no Microsoft 365. Consulte [Contatos](contacts.md) <br/> |Todos na organização podem exibir os contatos diretamente do Outlook, do Outlook na Web e de dispositivos móveis.  <br/> |Somente administradores podem criar e atualizar os contatos.  <br/> Nenhum campo personalizado é permitido (exemplo: data de nascimento, escola, agente de referência).  <br/> |
|**Contatos do SharePoint** <br/> |Use um aplicativo contatos do SharePoint para armazenar seus clientes ou contatos comerciais e eles aparecerão em seu site do SharePoint e aplicativos móveis do SharePoint. Confira [listas e aplicativos do SharePoint.](https://support.microsoft.com/office/0a1c3ace-def0-44af-b225-cfa8d92c52d7)  <br/> |Todos com acesso ao site de equipe do SharePoint e aos aplicativos móveis do SharePoint podem ver e atualizar esses contatos.  <br/> Permite campos personalizados, exibições, fluxos de trabalho e muito mais.  <br/> |Esses contatos não aparecem no Outlook ou pessoas no Microsoft 365.  <br/> Requer noções básicas sobre a infraestrutura do SharePoint.  <br/> |
|**Acessar contatos** <br/> |Use um aplicativo Contatos do Access para armazenar seus clientes ou contatos comerciais e eles aparecerão no Access ou em um site do SharePoint, se hospedados lá. Consulte [Aplicativos do Access.](https://support.microsoft.com/office/25f3ab3e-510d-44b0-accf-b976c0813e71)  <br/> |Todos com acesso ao Access ou ao site do SharePoint que hospeda o aplicativo podem ver e atualizar contatos.  <br/> Gerenciamento de contatos com aparência altamente personalizável e profissional criado com o Access.  <br/> |Você deve comprar o Microsoft Access ou mudar para um plano do Microsoft 365 que inclua o Access.  <br/> Requer noções básicas sobre o Microsoft Access e como criar aplicativos.  <br/> |
|**Dynamics CRM Online** <br/> |Use o Microsoft Dynamics CRM Online para armazenar e gerenciar seus clientes e possíveis oportunidades de negócios. Consulte [Dynamics CRM](https://dynamics.microsoft.com).  <br/> |Poderosos recursos de crm (gerenciamento de relacionamento com o cliente).  <br/> Integra-se ao Microsoft 365 para que você possa trabalhar em um só lugar.  <br/> |Inclui complexidade com integração e personalização para atender às suas necessidades.  <br/> É um custo significativamente maior do que qualquer outra opção de gerenciamento de contatos.  <br/> |
|**Aplicativo de terceiros** <br/> |Se nenhuma das opções acima lhe interessam, você pode considerar um aplicativo de terceiros na [Office Store](https://store.office.com) ou externo à Microsoft.  <br/> |Muitos provedores, recursos e preços para escolher para seus clientes comerciais.  <br/> |Nenhuma garantia de que ele está integrado ao Microsoft 365, exigindo que você trabalhe com dois serviços diferentes, logons etc.  <br/> |
