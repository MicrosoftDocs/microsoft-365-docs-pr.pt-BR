# <a name="set-up-your-domain"></a>Configurar seu domínio

Para começar a usar um domínio personalizado (contoso.com) com Microsoft 365, você precisa verificar seu domínio e configurar os registros DNS do seu domínio. 
  
Você pode adicionar e gerenciar registros DNS usando as ferramentas administrativas em seu host de domínio ou dar à Microsoft o controle de seus registros de domínio e os configuraremos para você.

## <a name="let-microsoft-365-manage-your-dns-records"></a>Deixe Microsoft 365 gerenciar seus registros DNS 

Saiba como alterar [os nameservers para configurar Microsoft 365 com qualquer registrador de domínios.](change-nameservers-at-any-domain-registrar.md)

## <a name="manage-your-own-dns-records"></a>Gerenciar seus próprios registros DNS

Para obter instruções, consulte [Create DNS records at any DNS hosting provider](create-dns-records-at-any-dns-hosting-provider.md)
