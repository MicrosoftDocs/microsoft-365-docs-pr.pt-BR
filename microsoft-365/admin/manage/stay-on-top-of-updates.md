# <a name="stay-on-top-of-changes"></a>Manter informado sobre as alterações

Com o Microsoft 365, você recebe novas atualizações de produto e recursos à medida que eles são disponibilizados em vez de atualizações agendadas meses ou anos. Como resultado, você e seus usuários poderão experimentar continuamente formas aprimoradas de trabalhar, em vez de esperar por uma atualização implementada em toda a empresa, que se torna cara e demorada. O desafio com um modelo desse tipo é conseguir acompanhar todas as alterações e atualizações. Veja a seguir algumas maneiras de permanecer sobre as atualizações do Microsoft 365 em sua organização.

## <a name="stay-on-top-of-microsoft-365-changes"></a>Mantenha-se atualizado com as alterações da Microsoft 365

||||
|:-----|:-----|:-----|
|**Recurso** <br/> |**Descrição** <br/> |**Como usar** <br/> |
|**Centro de mensagens** <br/> |Saiba mais sobre comunicados de serviço oficiais e alterações de recursos. Você pode ler essas mensagens no centro de administração do Microsoft 365, no aplicativo móvel do administrador ou receber um resumo semanal no email. Compartilhe essas mensagens com outras pessoas em sua organização quando você vir uma mensagem de que outra pessoa deve agir. Você também pode usar a API do Serviço de Comunicações para recuperar mensagens.  <br/> |Entre no [Centro de administração](../admin-overview/about-the-admin-center.md) ou [no aplicativo móvel do administrador](../admin-overview/admin-mobile-app.md). Selecione **Health** \> **centro de mensagens**de integridade. Selecione uma mensagem para ler ou compartilhar.  <br/> Altere os serviços que você vê para ver as mensagens ou aceitar o resumo semanal escolhendo **Editar preferências** no centro de administração. Também é aí que você pode recusar o resumo semanal.  <br/> [Visão geral do centro de mensagens do Microsoft 365](message-center.md) <br/> |
|**Lançamento direcionado** <br/> |Inscreva-se para você e um grupo seleto de pessoas em sua organização receberem o lançamento direcionado. Obtenha as atualizações mais recentes da Microsoft 365 antes de qualquer outra pessoa e, em seguida, informe ou treine seus usuários sobre a nova experiência.  <br/> |Entre no [Centro de administração](../admin-overview/about-the-admin-center.md) ou [no aplicativo móvel do administrador](../admin-overview/admin-mobile-app.md).
