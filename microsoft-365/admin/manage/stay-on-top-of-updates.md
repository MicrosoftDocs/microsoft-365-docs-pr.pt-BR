# <a name="stay-on-top-of-changes"></a>Manter informado sobre as alterações

Com Microsoft 365, você recebe novas atualizações de produtos e recursos à medida que elas se tornam disponíveis em vez de atualizações agendadas com meses ou anos de diferença. Como resultado, você e seus usuários poderão experimentar continuamente formas aprimoradas de trabalhar, em vez de esperar por uma atualização implementada em toda a empresa, que se torna cara e demorada. O desafio com um modelo desse tipo é conseguir acompanhar todas as alterações e atualizações. Aqui estão algumas maneiras de você ficar por dentro das atualizações Microsoft 365 em sua organização.

## <a name="stay-on-top-of-microsoft-365-changes"></a>Mantenha-se atualizado sobre as alterações do Microsoft 365

||||
|:-----|:-----|:-----|
|**Recurso** <br/> |**Descrição** <br/> |**Como usar** <br/> |
|**Centro de mensagens** <br/> |Saiba mais sobre anúncios de serviço oficiais e alterações de recursos. Você pode ler essas mensagens no centro de administração Microsoft 365 administrador, o aplicativo móvel do administrador ou receber um resumo semanal por email. Compartilhe essas mensagens com outras pessoas em sua organização quando vir uma mensagem em que outra pessoa deve agir. Você também pode usar a API do Serviço de Comunicações para recuperar mensagens.  <br/> |Entre no [Centro de administração](../admin-overview/about-the-admin-center.md) ou [no aplicativo móvel do administrador](../admin-overview/admin-mobile-app.md). Selecione **Centro de** Mensagens de \> **Saúde**. Selecione uma mensagem para ler ou compartilhar.  <br/> Altere os serviços sobre os quais você vê mensagens ou opte pelo resumo semanal escolhendo **Editar preferências** no centro de administração. Isso também é onde você pode optar por não participar do resumo semanal.  <br/> [Visão geral do centro de Microsoft 365 de mensagens](message-center.md) <br/> |
|**Lançamento direcionado** <br/> |Inscreva-se para você e um grupo seleto de pessoas em sua organização receberem o lançamento direcionado. Receba as atualizações Microsoft 365 mais recentes antes de todos os outros e informe ou treine seus usuários sobre a nova experiência.  <br/> |Entre no [Centro de administração](../admin-overview/about-the-admin-center.md) ou [no aplicativo móvel do administrador](../admin-overview/admin-mobile-app.md). Selece **Configurações** \>  \> **Preferências de versão do perfil da organização.** Saiba mais sobre o [Lançamento direcionado](release-options-in-office-365.md).  <br/> |
|**Mapa** <br/> |Visite o Microsoft 365 roteiro para saber mais sobre os recursos que foram lançados, estão sendo lançados, estão em desenvolvimento, foram cancelados ou lançados anteriormente. O roteiro é o site oficial para Microsoft 365 atualizações e alterações.  <br/> |Visite o [Microsoft 365 roteiro](https://www.microsoft.com/microsoft-365/roadmap) com frequência e saiba mais sobre atualizações e versões planejadas.  <br/> |
|**Blogs** e **Comunidade** <br/> |Visite Office Blogs, Microsoft Community e Microsoft Tech Community para saber mais sobre as alterações no Microsoft 365 e compartilhar experiências com outros usuários.  <br/> |Visite os [Blogs do Office](https://www.microsoft.com/en-us/microsoft-365/blog/). Visite a [Microsoft Community](https://answers.microsoft.com). Visite a [Microsoft Tech Community](https://techcommunity.microsoft.com).  <br/> |

> [!NOTE]
> Você precisa ser um administrador global para fazer alterações nas preferências de versão.
