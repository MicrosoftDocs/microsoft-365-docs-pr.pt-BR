# <a name="message-center"></a>Centro de mensagens


Para acompanhar as alterações futuras, incluindo recursos novos e alterados, manutenção planejada ou outros comunicados importantes, vá para o <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Centro de mensagens.</a> 
  
Para abrir o Centro de mensagens:

::: moniker range="o365-worldwide"

- No centro de administração, vá para o **Centro de** > <a href="https://go.microsoft.com/fwlink/p/?linkid=2070717" target="_blank">Mensagens de Saúde.</a>

::: moniker-end

::: moniker range="o365-germany"

- No centro <a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">de administração,</a>vá para o **Centro de** Mensagens > **de Saúde.**

::: moniker-end

::: moniker range="o365-21vianet"
 
- No centro <a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">de administração,</a>vá para o **Centro de** Mensagens > **de Saúde.**

::: moniker-end

Você também pode usar o aplicativo administrador do [Microsoft 365](https://go.microsoft.com/fwlink/p/?linkid=627216) em seu dispositivo móvel para exibir o Centro de mensagens, que é uma ótima maneira de se manter atualizado com as notificações por push.

### <a name="messages"></a>Mensagens

O centro de mensagens apresenta uma exibição de todas as mensagens ativas em um formato de tabela. Por padrão, ele mostra a mensagem mais recente na parte superior da lista. Você pode selecionar uma guia para  ver todas as mensagens **ativas,** mensagens de alta importância, mensagens não **lidas** e **mensagens descartadas.** E você pode usar **o** menu suspenso Filtro para ver todas as mensagens em uma categoria de mensagem específica.

Aqui está uma visão geral rápida das informações que você verá em cada coluna. Selecione qualquer um dos títulos de coluna para alterar a ordem de classificação.
  
|**Coluna**|**Descrição**|
|:-----|:-----|
|Marca de seleção  <br/> |Selecionar a marca de seleção na linha de título da coluna selecionará todas as mensagens exibidas no momento. Selecionar a marca de seleção ao lado de uma ou mais mensagens permite que você tome medidas nessas mensagens.  <br/> |
|Ícone do aplicativo <br/> |Ícones indicam o aplicativo ao qual a mensagem se aplica.<br/> |
|Título da mensagem  <br/> |Títulos de mensagem são breves descrições das alterações futuras. Se o título completo não for exibido, passe o cursor sobre ele e o título inteiro aparecerá em uma caixa pop-up.  <br/> |
|Mais opções <br/> |Mais opções permitem descartar uma mensagem, marcá-la como lida ou não lida ou compartilhá-la com outro administrador. Para restaurar uma mensagem descartada, selecione a guia Mensagens **descartadas,** marque a marca de seleção ao lado da mensagem e selecione **Restaurar**. <br/> |
|Atualização principal <br/> |Um ponto de exclamação nesta coluna indica uma atualização importante. <br/> |

### <a name="major-updates"></a>Atualizações principais

As principais atualizações podem ser revisadas selecionando a **guia** Alta prioridade na parte superior do Centro de Mensagens.

As atualizações principais são comunicadas com pelo menos 30 dias de antecedência quando uma ação é necessária e pode incluir:
  
- Alterações na produtividade diária, como caixa de entrada, reuniões, delegações, compartilhamento e acesso

- Alterações em temas, Web Parts e outros componentes que podem afetar recursos personalizados

- Aumenta ou diminui a capacidade visível, como armazenamento, número de regras, itens ou durações

- Alterações na identidade visual do produto que podem:

  - Causar confusão para o usuário final,

  - Result in changes to help desk processes and reference material, or

  - Alterar uma URL

- Um novo serviço ou aplicativo

- Alterações que exigem uma ação do administrador (exclusivamente para evitar ou corrigir problemas)

- Alterações no local em que seus dados estão armazenados
  
### <a name="preferences"></a>Preferências

Se a administração for distribuída em toda a organização, talvez você não queira ou precise ver postagens sobre todos os serviços do Microsoft 365. Cada administrador pode:

- Definir preferências que controlam quais mensagens são exibidas no Centro de mensagens.
- Filtrar mensagens
- De acordo com as preferências de email para receber um resumo semanal de todas as mensagens, emails apenas para atualizações importantes e emails para mensagens de privacidade de dados.  

::: moniker range="o365-worldwide"

1. Selecione **Editar preferências** na parte superior do Centro de mensagens.

2. Verifique se a caixa de seleção está marcada para cada serviço que você deseja monitorar. Des clear the check boxes for the services you want to filter out of your Message center view.

3. Os emails digest são ligado por padrão e enviados para seu endereço de email principal. Para parar de receber o resumo semanal, des clear **the Send a weekly email digest of my messages** check box. <br/><br/>A notificação por email para atualizações principais é um controle separado. Se você quiser receber avisos por email sobre as principais atualizações, verifique se a opção **Enviar-me emails** para as atualizações principais está selecionada. Des clear the check box to stop getting email about major updates. <br><br/>Marque a caixa de seleção ao lado de **Enviar-me emails** para mensagens de privacidade de dados a fim de receber notificações de email separadas das mensagens de privacidade de dados (as mensagens de privacidade de dados não são incluídas no resumo semanal). <br><br/>Você pode selecionar ou limpar seu endereço de email principal, mas não pode alterá-lo. Para especificar outros endereços de email para os quais o email de mensagem de privacidade de dados é enviado, marque a caixa de seleção Outros endereços de **email** e insira os outros endereços de email para os quais você deseja que o email seja enviado. Insira o endereço de email de um grupo do Microsoft 365 ou de uma lista de distribuição se mais de duas pessoas devem receber emails de privacidade de dados.

4. Selecione **Salvar** para manter suas alterações.
  
::: moniker-end

Usamos a tradução automática para exibir automaticamente as mensagens em seu idioma preferencial. Leia [as postagens de Tradução de Idioma](language-translation-for-message-center-posts.md) para o Centro de Mensagens para obter mais informações sobre como definir seu idioma.
