# <a name="share-calendars-with-external-users"></a>Compartilhar calendários com usuários externos

Às vezes, é necessário que os usuários agendem reuniões com pessoas de fora da organização. Para simplificar o processo de encontrar horários comuns de reunião, Microsoft 365 permite disponibilizar calendários para essas pessoas. São pessoas que precisam ver os horários de livre e ocupado dos usuários em sua organização, mas não têm contas de usuário para sua Microsoft 365 organização.

Você pode habilitar o compartilhamento de calendário para todos os usuários em sua organização no Microsoft 365 de administração. Depois que o compartilhamento for habilitado, os usuários poderão usar Outlook Web App compartilhar seus calendários com qualquer pessoa dentro ou fora da organização. As pessoas dentro da organização podem exibir o calendário compartilhado juntamente com seu próprio calendário. As pessoas de fora da organização receberão uma URL que poderá ser usada para exibir o calendário. Os usuários em sua organização decidem quando compartilhar e quanto compartilhar.

> [!NOTE]
> Se você quiser compartilhar agendas com uma organização que utiliza o Exchange Server 2013 (uma solução instalada no local), o administrador do Exchange precisará configurar uma relação de autenticação com a nuvem. Isso é conhecido como federação e deve atender aos requisitos mínimos de software. Consulte [Compartilhamento](/exchange/sharing-exchange-2013-help) para obter mais informações.
  
## <a name="enable-calendar-sharing-using-the-microsoft-365-admin-center"></a>Habilitar o compartilhamento de calendário usando o Microsoft 365 de administração

1. No centro de administração, vá **para** Configurações \> **Org Configurações**.

2. Na guia **Serviços,** selecione **Calendário**.
  
3. Na página **Calendário,** escolha se deseja permitir que os usuários compartilhem seus calendários com pessoas de fora da sua organização que tenham Microsoft 365 ou Exchange. Escolha se deseja permitir que usuários anônimos (usuários sem credenciais) acessem calendários por meio de um convite de email.

4. Escolha que tipo de informações de calendário disponibilizar aos usuários. Você pode permitir todas as informações ou limitá-la somente a hora ou hora, assunto e local somente.

## <a name="invite-people-to-access-calendars"></a>Convidar pessoas para acessarem os calendários

Depois que o compartilhamento é habilitado, os proprietários de calendários podem estender convites para usuários específicos. Para obter instruções, consulte [Sharing your calendar in Outlook Web App](https://support.microsoft.com/office/7ecef8ae-139c-40d9-bae2-a23977ee58d5).
