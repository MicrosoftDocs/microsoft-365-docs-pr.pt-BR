# <a name="manage-messages-in-message-center"></a>Gerenciar mensagens no Centro de Mensagens

As postagens do centro de mensagens são anúncios importantes sobre recursos novos e alterados. Este artigo fornece mais informações sobre o que você pode fazer com as mensagens para ajudar no gerenciamento de alterações em sua organização.
  
## <a name="read-a-message"></a>Ler uma mensagem

To go to the Message center, go to the [admin center](https://go.microsoft.com/fwlink/p/?linkid=2024339), and select **Health**  >  **Message center**. 

Selecione um título de mensagem para abri-lo no painel de leitura. Cada mensagem contém um resumo da alteração, uma instrução de como a alteração pode afetá-lo, informações sobre como você pode preparar sua organização para a alteração e (na maioria dos casos) um link de informações adicionais onde você pode saber mais. As informações adicionais continuam disponíveis, mesmo depois que uma mensagem expira.

## <a name="give-feedback-on-a-post"></a>Fazer comentários sobre uma postagem

No Centro de mensagens, você pode selecionar uma mensagem para ver detalhes.

Se você quiser fornecer comentários sobre a  mensagem, selecione o ícone Like ou **Dislike** e forneça comentários na caixa de texto exibida. Não forneça informações pessoais. Opcionalmente, você pode selecionar Não há problema em entrar em contato **comigo sobre esse comentário** e, em seguida, selecionar **Enviar.**

## <a name="share-a-message"></a>Compartilhar uma mensagem

Vê uma mensagem em que outra pessoa precisa agir? Você pode compartilhar o conteúdo da mensagem com qualquer usuário por email:
  
1. Selecione a mensagem para abri-la e, em seguida, selecione **Compartilhar**.
  
2. Para compartilhar a mensagem, insira até dois endereços de email separados por dois-pontos. Você pode enviar para endereços de email individuais e de grupo. Opcionalmente, você pode optar por receber uma cópia da mensagem por email (a mensagem irá para seu endereço de email principal) ou adicionar uma mensagem pessoal para fornecer aos destinatários mais contexto.
  
3. Selecione **Compartilhar** para enviar o email.
    
## <a name="get-a-link"></a>Obter um link

Precisa acompanhar outro administrador para garantir que ele esteja ciente de uma alteração e tomar medidas? Você pode gerar um link para compartilhar no email ou no sistema de mensagens instantâneas, por exemplo, que conectará o usuário diretamente a essa mensagem. A pessoa com quem você compartilha o link deve ser um administrador no Microsoft 365 para sua organização ou ela não terá acesso à mensagem vinculada.

::: moniker range="o365-worldwide"

1. Selecione a mensagem para abri-la.

2. Selecione o **link Copiar para o ícone da Área de** Transferência (link).

3. Permita que a página da Web acesse sua Área de Transferência. Um link direto é copiado para a área de transferência.

4. Use Ctrl+V ou clique com o botão direito do mouse e selecione **Colar** para inserir o link.

::: moniker-end

::: moniker range="o365-germany"

1. Selecione a mensagem para abri-la.

2. Choose **More** \> **Get a link** from the action bar.

3. Permita que a página da Web acesse sua Área de Transferência. Um link direto é copiado para a área de transferência.

4. Use Ctrl+V ou clique com o botão direito do mouse e selecione **Colar** para inserir o link.

::: moniker-end

::: moniker range="o365-21vianet"

1. Selecione a mensagem para abri-la.

2. Choose **More** \> **Get a link** from the action bar.

3. Permita que a página da Web acesse sua Área de Transferência. Um link direto é copiado para a área de transferência.

## <a name="read-and-unread-states"></a>Estados lidos e não lidos

Qualquer mensagem no Centro de mensagens não lida aparecerá em negrito. Abrir uma mensagem marca como lida. Você pode marcar uma mensagem como não lida.

::: moniker range="o365-worldwide"

1. Na página principal do centro de mensagens, selecione uma mensagem.

2. Selecione o **ícone Marcar como não** lido (envelope) na parte superior da mensagem.
  
::: moniker-end

::: moniker range="o365-germany"

1. Selecione a mensagem para abri-la.

2. Selecione **Mais** \> **Marca como não lida.**

3. Feche a mensagem.

::: moniker-end

::: moniker range="o365-21vianet"

1. Selecione a mensagem para abri-la.

2. Selecione **Mais** \> **Marca como não lida.**

3. Feche a mensagem.

::: moniker-end

## <a name="archive-and-restore"></a>Arquivar e restaurar

Se você vir uma mensagem que não pertence a você, ou talvez você já atuou nele, poderá arquivar a mensagem para removê-la da exibição centro de mensagens. O modo de exibição que você vê no Centro de mensagens é específico para sua conta de usuário, portanto, arquiva-la do seu modo de exibição não afeta outros administradores.

::: moniker range="o365-worldwide"

- Na página principal do Centro de mensagens, selecione uma mensagem e, em seguida, selecione **Arquivar** acima da lista de mensagens.

- Abra a mensagem e selecione **Arquivar** na parte inferior da mensagem.

Precisa receber uma mensagem arquivada de volta? Não há problema.
  
1. Selecione a **guia Mensagens Arquivadas** na parte superior do Centro de Mensagens. Uma lista de mensagens arquivadas é exibida. 

2. Selecione a mensagem, selecione **Restaurar** e a mensagem será restaurada para mensagens ativas.

::: moniker-end

::: moniker range="o365-germany"

Há duas maneiras de arquivar uma mensagem.
  
- Selecione o X que aparece à direita da ID da Mensagem. A mensagem foi removida da sua lista. Ou,

- Abra a mensagem e selecione **Arquivar** na parte superior da mensagem.

Precisa receber uma mensagem descartada de volta? Não há problema.
  
1. Selecione o **filtro de mensagens arquivadas** **no** menu suspenso Exibições. Uma lista de mensagens arquivadas é exibida.

2. Selecione **Restaurar** à direita da ID da Mensagem e a mensagem será restaurada para as mensagens ativas.

::: moniker-end

::: moniker range="o365-21vianet"

Há duas maneiras de arquivar uma mensagem.
  
- Selecione o X que aparece à direita da ID da Mensagem. A mensagem foi removida da sua lista. Ou,

- Abra a mensagem e selecione **Arquivar** na parte superior da mensagem.

Precisa receber uma mensagem arquivada de volta? Não há problema.
  
1. Selecione o **filtro de mensagens arquivadas** **no** menu suspenso Exibições. Uma lista de mensagens arquivadas é exibida.

2. Selecione **Restaurar** à direita da ID da Mensagem e a mensagem será restaurada para as mensagens ativas.

::: moniker-end

## <a name="track-your-message-center-tasks-in-planner"></a>Acompanhe o centro de mensagens no Planner

Uma grande quantidade de informações acionáveis sobre as alterações nos serviços do Microsoft 365 chega no centro de mensagens da Microsoft 365. Pode ser difícil acompanhar quais alterações exigem que as tarefas sejam feitas, quando e por quem, e rastrear cada tarefa até a conclusão. Você também pode querer fazer uma anotação de algo e marcá-la para que seja verificada depois. Você pode fazer tudo isso e muito mais ao sincronizar suas mensagens do Centro de administração do Microsoft 365 para o Microsoft Planner. Para saber mais, confira [Controlar as tarefas do centro de mensagens no Planner.](https://docs.microsoft.com/office365/planner/track-message-center-tasks-planner)


Para ter uma visão geral do Centro de mensagens, confira [Centro de mensagens no Microsoft 365.](message-center.md) Ou, para saber como definir suas preferências de idioma para habilitar a tradução automática para postagens do Centro de mensagens, consulte Tradução de idioma para [postagens do Centro de mensagens.](language-translation-for-message-center-posts.md) Se você quiser programar uma maneira alternativa de obter informações de saúde do serviço em tempo real e comunicações do Centro de Mensagens, faça referência à Visão Geral da API de Comunicações do Serviço do [Microsoft 365.](https://go.microsoft.com/fwlink/p/?linkid=848507)
