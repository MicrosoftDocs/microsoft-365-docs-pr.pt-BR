# <a name="explaining-microsoft-365-groups-to-your-users"></a>Explicando Microsoft 365 grupos para seus usuários

Microsoft 365 Os grupos permitem configurar uma coleção de recursos para compartilhar, incluindo uma caixa de correio e calendário compartilhados, um site SharePoint com um bloco de anotações OneNote e um Microsoft Planner entre outros. Microsoft Teams também pode ser incluído ao criar um grupo ou pode ser adicionado posteriormente. Os recursos dos grupos de permissões são gerenciados por meio do grupo.

Os grupos podem ser criados criando qualquer um dos recursos compartilhados. A criação de um grupo Outlook resulta no mesmo resultado que a criação de um site de equipe SharePoint grupo ou um plano no Planner. Se os usuários são novos Microsoft 365 grupos, eles podem não perceber isso. Isso pode levar a confusão para seus usuários e a possibilidade de criar recursos duplicados. (Por exemplo, alguém pode criar um site SharePoint para colaboração de documentos e, posteriormente, criar uma instância separada do Planner, sem perceber que o Planner já estava disponível como parte do grupo.)

Como os grupos podem ser criados de várias maneiras, recomendamos treinar seus usuários para usar o método que melhor se ajuste à sua organização:

- Se a sua organização fizer a maior parte da comunicação usando email, instrua os usuários a criar grupos Outlook.
- Se sua organização usa muito SharePoint ou está migrando do SharePoint local, instrua os usuários a criarem sites SharePoint equipe para colaboração.
- Se sua organização tiver implantado Teams, instrua os usuários a criar uma equipe quando eles precisam de um espaço de colaboração.

[![desc de imagem ](../../media/03.png)](../../media/03.png#lightbox)

Se você treinar seus usuários para sempre usar o método de criação de grupo que mais se alinha à sua maneira de trabalhar quando eles precisam de um espaço para colaborar com outras pessoas, você pode ajudar a evitar confusão e duplicação de recursos. À medida que os usuários se tornam mais experientes, eles entenderão melhor a coleção de serviços que vêm com um grupo e que diferentes métodos de criação levam ao mesmo resultado.

Você pode usar o modelo [Microsoft 365 Grupos para Empresas -](https://www.microsoft.com/download/details.aspx?id=102396) PowerPoint como ponto de partida para apresentações de treinamento para seus usuários.
