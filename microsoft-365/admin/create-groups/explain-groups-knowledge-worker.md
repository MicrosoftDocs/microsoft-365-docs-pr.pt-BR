# <a name="explaining-microsoft-365-groups-to-your-users"></a>Explicando os Grupos do Microsoft 365 para seus usuários

Os Grupos do Microsoft 365 permitem configurar um conjunto de recursos para compartilhar, incluindo uma caixa de correio e calendário compartilhados, um site do SharePoint com um bloco de anotações do OneNote e um Microsoft Planner entre outros. O Microsoft Teams também pode ser incluído quando você cria um grupo ou pode ser adicionado posteriormente. Os recursos dos grupos de permissões são gerenciados por meio do grupo.

Os grupos podem ser criados criando qualquer um dos recursos compartilhados. Criar um grupo no Outlook gera o mesmo resultado que criar um site de equipe do SharePoint conectado ao grupo ou um plano no Planner. Se os usuários não são novos nos Grupos do Microsoft 365, eles podem não perceber isso. Isso pode causar confusão para os usuários e a possibilidade de criar recursos duplicados. (Por exemplo, alguém pode criar um site do SharePoint para colaboração em documentos e, mais tarde, criar uma instância separada do Planner, sem perceber que o Planner já estava disponível como parte do grupo.)

Como os grupos podem ser criados de várias maneiras, recomendamos que os usuários usem o método mais eficaz para a sua organização:

- Se a sua organização faz a maior parte da comunicação usando email, instrua os usuários a criar grupos no Outlook.
- Se sua organização usa intensamente o SharePoint ou está migrando do SharePoint local, instrua seus usuários a criar sites de equipe do SharePoint para colaboração.
- Se sua organização implantou o Teams, instrua os usuários a criar uma equipe quando eles precisam de um espaço de colaboração.

[![desc de imagem ](../../media/03.png)](../../media/03.png#lightbox)

Se você treinar seus usuários para sempre usar o método de criação de grupo que mais se alinha à sua maneira de trabalhar quando eles precisam de um espaço para colaborar com outras pessoas, você pode ajudar a evitar confusão e duplicação de recursos. À medida que os usuários se tornam mais experientes, eles entendem melhor a coleção de serviços que vêm com um grupo e que diferentes métodos de criação levam ao mesmo resultado.

Você pode usar o modelo De usuário do PowerPoint dos Grupos [do Microsoft 365](https://www.microsoft.com/download/details.aspx?id=102396) para Empresas como ponto de partida para apresentações de treinamento para seus usuários.
