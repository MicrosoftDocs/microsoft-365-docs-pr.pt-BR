# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Obter a versão mais recente da análise de uso do Microsoft 365

O aplicativo de modelo pode ser atualizado com novos dados ou novas visualizações várias vezes por ano. Sua instância existente continuará funcionando, mas se você quiser obter a versão mais recente, uma nova instância deverá ser criada e qualquer personalização deverá ser aplicada à nova instância. Consulte [Habilitar análise de uso do Microsoft 365](enable-usage-analytics.md).
