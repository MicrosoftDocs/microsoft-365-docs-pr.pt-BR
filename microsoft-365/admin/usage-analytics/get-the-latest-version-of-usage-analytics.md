# <a name="get-the-latest-version-of-microsoft-365-usage-analytics"></a>Obter a versão mais recente da análise de uso do Microsoft 365

O aplicativo de modelo pode ser atualizado com novos dados ou novas visualizações várias vezes por ano. Sua instância existente continuará a funcionar, mas se você quiser obter a versão mais recente, uma nova instância deve ser criada e qualquer personalização deve ser aplicada à nova instância. Consulte [Habilitar a análise de uso do Microsoft 365.](enable-usage-analytics.md)
