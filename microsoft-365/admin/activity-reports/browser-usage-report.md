# <a name="microsoft-365-reports-in-the-admin-center---microsoft-browser-usage"></a>Relatórios do Microsoft 365 no centro de administração - uso do navegador da Microsoft

O painel Relatórios  do Microsoft 365 mostra uma visão geral de atividades em todos os produtos em sua organização. Ele permite que você faça uma pesquisa em relatórios de nível de produto individuais para dar uma visão mais granular sobre as atividades em cada produto. Confira o tópico [Visão geral de relatórios](activity-reports.md). No relatório de uso do navegador da Microsoft, você pode obter informações sobre o Internet Explorer, o Microsoft Edge Legacy e o novo uso do Microsoft Edge. O relatório de uso baseia-se nos serviços online do Microsoft 365 acessados por meio de um navegador da Microsoft.

 > [!NOTE]
 > Você deve ser um administrador global, leitor global ou leitor de relatórios no Microsoft 365 ou administrador do Exchange, SharePoint ou Skype for Business para ver relatórios.

## <a name="how-to-get-to-the-microsoft-browser-usage-report"></a>Como chegar ao relatório de uso do navegador da Microsoft

1. No centro de administração do, vá para a página **Relatórios** \> <a href="https://go.microsoft.com/fwlink/p/?linkid=2074756" target="_blank">Uso</a>. 
2. Na página inicial do painel, clique no botão **Exibir mais** no cartão de uso do navegador da Microsoft.

## <a name="interpret-the-microsoft-browser-usage-report"></a>Interpretar o relatório de uso do navegador da Microsoft

![Relatório de uso do navegador da Microsoft](../../media/95557c88-24ee-417d-a828-96ba00b17aaf.png)

|Item|Descrição|
 |:-----|:-----|
 |1. <br/> |O **relatório de uso** do navegador da Microsoft pode ser exibido para tendências nos últimos 7 dias, 30, 90 ou 180 dias.  <br/> |
 |2. <br/> |Os dados em cada relatório geralmente abrangem até os últimos sete dias. <br/> |
 |3. <br/> |O **gráfico Usuários ativos** diários mostra a contagem diária de usuários para o Microsoft Edge, o Microsoft Edge Legacy e o Internet Explorer quando usado para acessar os serviços do Microsoft 365. <br/> |
 |4.<br/>|O **gráfico Usuários ativos** mostra o número total de usuários que usam o Microsoft Edge, o Microsoft Edge Legacy e o Internet Explorer quando usados para acessar os serviços do Microsoft 365 durante o período de tempo selecionado.<br/>|
 |5.<br/>|A tabela mostra o detalhamento dos dados no nível do usuário. Você pode adicionar ou remover colunas da tabela. <br/><br/>**Username** é o endereço de email do usuário que se conectou aos serviços do Microsoft 365 usando navegadores da Microsoft.<br><br/>**O Microsoft Edge** usado mostra uma marca de escala se o usuário usou o Microsoft Edge para se conectar aos serviços do Microsoft 365.<br/><br/>**O Microsoft Edge Legacy** usado mostra uma marca de escala se o usuário usou o Microsoft Edge Legacy para se conectar aos serviços do Microsoft 365.<br/><br/>**O Internet Explorer** usado mostra uma marca de escala se o usuário usou o Internet Explorer para se conectar aos serviços do Microsoft 365. |
 |6.<br/>|Selecione o **ícone Escolher colunas** para adicionar ou remover colunas do relatório.|
 |7.<br/>|Você também pode exportar os dados do relatório para um arquivo .csv do Excel selecionando o link **Exportar.** Isso exporta dados para todos os usuários e permite que você faça agregação, classificação e filtragem simples para análise posterior. Se você tiver menos de 100 usuários, poderá classificar e filtrar dentro da tabela no próprio relatório. Se você tiver mais de 100 usuários, para filtrar e classificar, será necessário exportar os dados.|
