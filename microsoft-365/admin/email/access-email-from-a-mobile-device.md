# <a name="options-for-accessing-email-from-your-mobile-device"></a>Opções para acessar emails do seu dispositivo móvel

Sua organização define configurações que controlam quais aplicativos móveis você pode usar para acessar sua conta de trabalho para email, calendário e contatos.

O Outlook para Android e os aplicativos móveis do Outlook para iOS foram projetados para seu email de trabalho ou escola. Esses aplicativos [](https://support.microsoft.com/office/f445ad7f-02f4-4294-a82e-71d8964e3978)fornecem recursos como a Caixa de Entrada Destaques, que mantém emails importantes na frente e no centro, e o Assistente de Agendamento, [](https://support.microsoft.com/office/scheduling-made-easy-in-outlook-mobile-11c5bee5-d78a-4a2b-80c2-2b386ddb4470)que ajuda você a encontrar horários de reunião que funcionam com colegas e simplesmente encontra salas de conferência disponíveis.
  
## <a name="end-user-experience"></a>Experiência do usuário final

### <a name="outlook-only"></a>Somente Outlook

Sua organização exige que você use o Outlook para Android ou o Outlook para aplicativos móveis para iOS para acessar email, calendário e contatos da empresa. Seus dados começarão a sincronizar depois que você baixar e instalar o Outlook para Android ou o Outlook para iOS.

![Exemplo de email para usar o Outlook para sincronizar emails](../../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)

Confira Otimizar [o aplicativo móvel do Outlook para seu telefone iOS](https://support.microsoft.com/office/de075b19-b73c-4d8a-841b-459982c7e890) ou Android para obter mais recursos do Outlook. E vá para [o Outlook para iOS e a](https://support.microsoft.com/office/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6) Central de Ajuda do Android se você tiver algum problema.

### <a name="any-email-app"></a>Qualquer aplicativo de email

Sua organização recomenda que você use o aplicativo móvel Outlook para Android ou iOS para acessar sua conta comercial ou de estudante para email, calendário e contatos. Se você não tomar nenhuma ação nas próximas horas, seu email começará automaticamente a sincronização. Se você optar por usar o aplicativo móvel outlook para Android ou iOS, seus dados serão sincronizados apenas com o Outlook para Android ou outlook para aplicativo móvel iOS. Se você optar por continuar usando o aplicativo de terceiros, seus dados começarão a ser sincronizados instantaneamente.
