# <a name="options-for-accessing-email-from-your-mobile-device"></a>Opções para acessar emails do seu dispositivo móvel

Sua organização configura configurações que controlam quais aplicativos móveis você pode usar para acessar sua conta de trabalho para email, calendário e contatos.

As Outlook para Android e Outlook para aplicativos móveis iOS foram projetadas para seu email de trabalho ou de estudante. Esses aplicativos fornecem recursos como [Caixa](https://support.microsoft.com/office/f445ad7f-02f4-4294-a82e-71d8964e3978)de Entrada Destaque , que mantém emails importantes na frente e no [centro,](https://support.microsoft.com/office/scheduling-made-easy-in-outlook-mobile-11c5bee5-d78a-4a2b-80c2-2b386ddb4470)e Assistente de Agendamento , o que ajuda você a encontrar horários de reunião que funcionam com colegas e simplesmente encontra salas de conferência disponíveis.
  
## <a name="end-user-experience"></a>Experiência do usuário final

### <a name="outlook-only"></a>Outlook Somente

Sua organização exige que você use o Outlook para Android ou Outlook para aplicativos móveis iOS para acessar email, calendário e contatos da empresa. Seus dados começarão a sincronizar depois que você baixar e instalar Outlook para Android ou Outlook para iOS.

![Exemplo de email para usar Outlook sincronizar emails](../../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)

Confira Otimizar [o aplicativo móvel Outlook para seu telefone iOS](https://support.microsoft.com/office/de075b19-b73c-4d8a-841b-459982c7e890) ou Android para obter mais Outlook recursos. E vá para o [Outlook para iOS e a Central](https://support.microsoft.com/office/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6) de Ajuda do Android se você tiver problemas.

### <a name="any-email-app"></a>Qualquer aplicativo de email

Sua organização recomenda que você use o Outlook para Android ou aplicativo móvel iOS para acessar sua conta comercial ou escolar para email, calendário e contatos. Se você não tomar nenhuma ação nas próximas horas, seu email começará automaticamente a sincronizar. Se você optar por usar o Outlook para android ou aplicativo móvel iOS, seus dados serão sincronizados apenas com o Outlook para Android ou Outlook para aplicativo móvel iOS. Se você optar por continuar usando o aplicativo de terceiros, seus dados começarão a sincronizar instantaneamente.
