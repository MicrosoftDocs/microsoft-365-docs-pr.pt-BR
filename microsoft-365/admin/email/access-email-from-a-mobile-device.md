# <a name="options-for-accessing-email-from-your-mobile-device"></a>Opções para acessar o email do seu dispositivo móvel

Sua organização definiu configurações que controlam os aplicativos móveis que você pode usar para acessar sua conta de trabalho por email, calendário e contatos.
  
Os aplicativos móveis do Outlook para Android e Outlook para iOS são projetados para seu email escolar ou de estudante, e são os únicos aplicativos a fornecer recursos como [caixa de entrada destaques](https://support.office.com/article/f445ad7f-02f4-4294-a82e-71d8964e3978.aspx), que mantém emails importantes e centralizados e o [Assistente de agendamento](https://go.microsoft.com/fwlink/?linkid=873406), que ajuda você a encontrar horários de reunião que trabalham com colegas e simplesmente localiza salas de conferência disponíveis.
  
## <a name="end-user-experience"></a>Experiência do usuário final

 **Somente Outlook**
  
Sua organização requer que você use o Outlook para Android ou o Outlook para aplicativos móveis do iOS para acessar email, calendário e contatos da empresa. Os dados começarão a ser sincronizados depois que você baixar e instalar o Outlook para Android ou o Outlook para iOS.
  
![Email de exemplo para usar o Outlook para sincronizar email](../media/798d942a-4181-4dcb-8039-cd9f2edd9723.png)
  
Confira [otimizar o aplicativo móvel do Outlook para seu telefone Ios ou Android](https://support.office.com/article/de075b19-b73c-4d8a-841b-459982c7e890.aspx) para mais recursos do Outlook. E vá para o [Outlook para IOS e o centro de ajuda do Android](https://support.office.com/article/cd84214e-a5ac-4e95-9ea3-e07f78d0cde6.aspx) se você tiver problemas. 
  
 **Qualquer aplicativo de email**
  
Sua organização recomenda que você use o Outlook para Android ou o aplicativo móvel iOS para acessar sua conta corporativa ou de estudante do Office 365 para email, calendário e contatos. Se você não realizar nenhuma ação nas próximas horas, seu email começará automaticamente a sincronização. Se você optar por usar o aplicativo móvel do Outlook para Android ou iOS, seus dados só serão sincronizados com o aplicativo móvel do Outlook para Android ou Outlook para iOS. Se você optar por continuar usando o aplicativo de terceiros, seus dados serão iniciados para sincronizar instantaneamente.
