# <a name="difference-between-the-setup-wizard-and-the-setup-page"></a>Diferença entre o assistente de configuração e a página de instalação

O Microsoft 365 oferece duas experiências de configuração: 

- Configuração inicial usando o assistente de configuração
- Configuração contínua e avançada usando a página **Configuração**

O assistente de configuração fornece um guia passo a passo para configurar a configuração básica do Microsoft 365. Depois de concluir a configuração inicial, você pode ir para a página De instalação para concluir a configuração e a configuração dos serviços que vêm com suas assinaturas. 

## <a name="use-the-setup-wizard-to-complete-initial-setup-tasks"></a>Usar o assistente de configuração para concluir as tarefas iniciais de instalação

Para configurar sua conta, vá para  o centro de [administração,](https://go.microsoft.com/fwlink/p/?linkid=2024339)selecione  Instalação no painel de navegação esquerdo e selecione Configuração Guiada na página **Configuração.**

![Iniciar o assistente de configuração do Microsoft 365 Apps for Business](../../media/o365b-guided-setup.png)

O assistente de configuração orienta você pelas seguintes etapas:

1. Instale os aplicativos do Microsoft 365 no computador.

2. Escolha e verifique seu domínio, como contoso.com.

3. Adicione novos usuários e atribua licenças a eles para que eles possam baixar e instalar aplicativos do Microsoft 365.

4. Conecte seu domínio.

## <a name="use-the-setup-page-to-complete-and-manage-your-configuration"></a>Usar a página Instalação para concluir e gerenciar sua configuração

Para acessar a **página Configuração** no centro [de administração,](https://go.microsoft.com/fwlink/p/?linkid=2024339)selecione **Instalação** no painel de navegação. Com base nos produtos que você comprou, os recursos que você definiu e sua função de administrador, tarefas e informações relacionadas do Microsoft 365 são destacadas aqui.

Você verá a lista completa de tarefas de instalação organizadas em categorias lógicas, incluindo aquelas concluídas no assistente de configuração.

![Página de instalação do Microsoft 365 para empresas](../../media/o365b-setup-page.png)

Escolha  Exibir qualquer tarefa para obter informações rápidas, como descrição da tarefa, impacto do usuário, pré-requisitos, esforço para implementar e estatísticas de segurança e adoção para ajudá-lo a entender as consequências e o impacto antes de prosseguir.

You'll also see the status of the task (**Started**, **Not started yet**, or **Completed**). Se você estiver trabalhando com várias tarefas, trabalhando em tarefas ao longo de vários dias ou se houver vários administradores trabalhando em tarefas, poderá acompanhar a conclusão vendo rapidamente quais tarefas foram concluídas e quais ainda exigem atenção. 

Para acessar artigos abrangentes sobre os recursos que você está configurando, selecione qualquer um dos links **Saiba mais.** A coleção de tarefas está sempre aqui,  para que você possa retornar à página De instalação a qualquer momento para explorar ainda mais os recursos.

Quando você estiver pronto para concluir uma tarefa, selecione **Começar** a passar pelo processo de configuração. Depois de concluir uma tarefa, o **botão Começar** muda para um botão Gerenciar, permitindo que você gerencie **a** tarefa, conforme necessário.

![Exibição de tarefas mostrando informações rápidas](../../media/o365b-at-a-glance.png)
