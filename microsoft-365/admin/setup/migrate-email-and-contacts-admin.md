# <a name="migrate-email-and-contacts-to-microsoft-365"></a>Migrar email e contatos para o Microsoft 365

Importe ou migre emails do Gmail ou de outro provedor de email para o Microsoft 365.
  
 **Quer ajuda com isso?**  [Entre em contato com o suporte da Microsoft 365 para empresas](../contact-support-for-business-products.md) . 
  
Você precisa usar uma versão do Outlook instalada na sua área de trabalho para esta tarefa. O Outlook está incluído na maioria dos [planos](https://go.microsoft.com/fwlink/p/?LinkId=723731)do Microsoft 365.
  
### <a name="migrate-gmail-to-microsoft-365"></a>Migrar do Gmail para o Microsoft 365

Siga estas etapas para importar ou migrar emails, contatos e calendário do Gmail para o Outlook com o Microsoft 365:
  
- [Importar o Gmail para o Outlook](https://support.microsoft.com/office/20fdb8f2-fed8-4b14-baf0-bf04b9c44bf7)
    
- [Importar contatos para o Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
- [Importar o Google Calendar](https://support.microsoft.com/office/098ed60c-936b-41fb-83d6-7e3786437330)
    
> [!VIDEO https://www.microsoft.com/videoplayer/embed/c1e45708-9a92-475b-910b-f5aa40614d92?autoplay=false]
  
### <a name="import-outlook-pst-files-to-microsoft-365-desktop"></a>Importar arquivos pst do Outlook para o Microsoft 365 (Desktop)

Siga estas etapas para exportar emails, contatos e calendários do Outlook para um arquivo. pst e, em seguida, importe esse arquivo para o Outlook com o Microsoft 365:
  
1. [Exportar email, contatos e calendário](https://support.microsoft.com/office/14252b52-3075-4e9b-be4e-ff9ef1068f91)
    
2. [Importar email, contatos e calendário](https://support.microsoft.com/office/431a8e9a-f99f-4d5f-ae48-ded54b3440ac)
    
Se você quiser apenas contatos, siga estas etapas:
  
1. [Exportar contatos do Outlook](https://support.microsoft.com/office/10f09abd-643c-4495-bb80-543714eca73f)
    
2. [Importar contatos para o Outlook](https://support.microsoft.com/office/bb796340-b58a-46c1-90c7-b549b8f3c5f8)
    
Para iniciar o processo, abra o Outlook e escolha **arquivo** \> **abrir &amp; Exportar** \> **importar/exportar**.

## <a name="see-other-email-accounts-in-outlook"></a>Confira outras contas de email no Outlook

Deseja apenas ver seus emails de outro provedor (como o Gmail, o Yahoo ou o Live.com) no Outlook? Não é necessário importar ou migrar. É possível configurar o Outlook ou o Outlook Web App para que você possa acessar suas outras contas no mesmo local da caixa de correio do Microsoft 365 e enviar, receber e ler emails enviados para essas contas.
  
### <a name="outlook-desktop"></a>Outlook (área de trabalho)

Adicione uma conta, como sua conta do Gmail privado, ao Outlook.
  
- Abra o Outlook e vá para **arquivo** \> **adicionar conta**.
    
Precisa de etapas mais detalhadas? Consulte [Adicionar uma conta](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b).
  
[![Captura de tela mostrando a página informações de conta do Outlook no modo de exibição Backstage.](../../media/6a7fa106-1077-4351-9fe2-8eb00918b40a.png)](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b)
  
## <a name="multiple-mailboxes-admins-can-bulk-import-email-contacts-and-calendars-to-microsoft-365"></a>Várias caixas de correio: os administradores podem importar email, contatos e calendários em massa para a Microsoft 365

Dependendo do sistema de email de origem, você pode escolher entre vários métodos de migração em massa. Leia [maneiras de migrar várias contas de email para a Microsoft 365](https://docs.microsoft.com/Exchange/mailbox-migration/mailbox-migration) para decidir qual método funciona para você.
