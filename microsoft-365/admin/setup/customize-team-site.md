# <a name="customize-your-team-site-for-file-storage-and-sharing"></a>Personalizar seu site de equipe para armazenamento e compartilhamento de arquivos

Se você tiver uma pequena empresa com alguns funcionários, uma das melhores maneiras de configurar e facilitar o compartilhamento de arquivos e a colaboração online é usar o OneDrive e um site de equipe SharePoint em conjunto. Recomendamos a criação de um site de equipe conectado a um Microsoft 365 grupo. Se você quiser usar o chat, pode criar esse site criando uma equipe em Microsoft Teams. Para obter mais informações sobre Teams, consulte [Vídeo: O que é Microsoft Teams?](https://support.microsoft.com/office/b98d533f-118e-4bae-bf44-3df2470c2b12). Se você não quiser usar Teams, você pode criar um site de equipe a partir da página SharePoint inicial. 
  
[O que é um site de equipe do SharePoint?](https://support.microsoft.com/office/75545757-36c3-46a7-beed-0aaa74f0401e)
  
[Criar um site de equipe em SharePoint](https://support.microsoft.com/office/ef10c1e7-15f3-42a3-98aa-b5972711777d)
  
Aqui estão algumas ideias e links para ajudá-lo a personalizar, organizar e gerenciar seu site de equipe.
  
 
## <a name="customize-your-team-site"></a>Personalizar seu site de equipe

Para dar mais visibilidade aos dados em um site de equipe, você pode personalizar a página inicial do SharePoint para ajudar você e seus funcionários a obter as informações de que você precisa rapidamente. Por exemplo, você pode ter links para pastas de OneDrive de funcionários e links para Documentos, Contatos, Tarefas, Calendários e um bloco de anotações OneNote onde você armazena agendas de reunião e anotações.
  
Para obter informações sobre como personalizar a aparência da página inicial, consulte Alterar a aparência [do seu SharePoint site](https://support.microsoft.com/office/06bbadc3-6b04-4a60-9d14-894f6a170818).
  
Melhorar a navegação em seu site com [Personalizar a navegação em seu site SharePoint site](https://support.microsoft.com/office/3cd61ae7-a9ed-4e1e-bf6d-4655f0bf25ca).
  
Para melhorar a capacidade de descoberta de um site, adicione-o como um link em destaque na página SharePoint inicial. Para obter informações, [consulte Adicionar links em destaque à página SharePoint inicial](/sharepoint/change-links-list-on-sharepoint-home-page).
  
Para adicionar uma imagem a uma página inicial do site de equipe, consulte Adicionar uma imagem ou arquivo de imagem [a uma SharePoint página](https://support.microsoft.com/office/4a9b0e98-c89a-4a41-8adb-b7750dccca16).
  
Para adicionar páginas ao seu site, consulte [Adicionar uma página a um site](https://support.microsoft.com/office/b3d46deb-27a6-4b1e-87b8-df851e503dec).
  
Para saber mais sobre as configurações do site de equipe, consulte [Manage your SharePoint team site settings](https://support.microsoft.com/office/8376034D-D0C7-446E-9178-6AB51C58DF42).
  
## <a name="work-with-document-libraries"></a>Trabalhar com bibliotecas de documentos

Um site de equipe inclui uma biblioteca de documentos que você pode começar a usar imediatamente para armazenamento de arquivos e gerenciamento de documentos.

Para começar a carregar, [consulte Upload uma pasta ou arquivos em uma biblioteca de documentos](https://support.microsoft.com/office/eb18fcba-c953-4d45-8d90-8da66edeacdb)
   
Saiba como editar, excluir, fazer check-out de arquivos e muito mais. Para obter mais informações, [consulte Trabalhar com arquivos em uma biblioteca de documentos](https://support.microsoft.com/office/a9d89171-1673-4892-9dd2-1ca52037dea2).
  
Para obter informações sobre como criar exibições de bibliotecas para exibir dados específicos, consulte [Create a custom view of a document library](https://support.microsoft.com/office/8f6b08e0-a9a0-4232-9b9b-b374a2ad3da7).
  
## <a name="work-with-lists-to-organize-data"></a>Trabalhar com listas para organizar dados

SharePoint inclui muitos modelos para listas que você pode adicionar ao seu site, como uma lista de links, um calendário, contatos ou tarefas.
  
Saiba como [criar uma lista em SharePoint](https://support.microsoft.com/office/0D397414-D95F-41EB-ADDD-5E6EFF41B083#ID0EAAGAAA=Online).
  
Para obter informações sobre como expandir suas listas com mais colunas, consulte [Create a column in a SharePoint list or library](https://support.microsoft.com/office/2b0361ae-1bd3-41a3-8329-269e5f81cfa2).
  
Se você estiver armazenar muitos dados em uma lista, consulte [Manage large lists and libraries in SharePoint](https://support.microsoft.com/office/B8588DAE-9387-48C2-9248-C24122F07C59).
  
Exibir um vídeo ao adicionar calendários ao seu site, consulte [Create a shared calendar](https://support.microsoft.com/office/61b96006-70e2-4535-a34f-ee4fc772f798).

## <a name="organize-sites-into-hubs"></a>Organizar sites em hubs

Seu site de equipe compartilha quase tudo com todos os membros do grupo associados ao site. Se você quiser ter um site separado para clientes específicos, pode criar sites de equipe adicionais e gerenciá-los usando SharePoint de hub. Veja mais informações.
  
Os sites de hub permitem que você adicione e associe sites. Para obter mais informações, consulte [O que é um site SharePoint hub?](https://support.microsoft.com/office/fe26ae84-14b7-45b6-a6d1-948b3966427f).
  
Pronto para criar um site de hub, consulte [Create a hub site in SharePoint Online](/sharepoint/create-hub-site).
  
Para saber como adicionar sites a um site de hub, consulte [Associate a SharePoint site with a hub site](https://support.microsoft.com/office/ae0009fd-af04-4d3d-917d-88edb43efc05).
  
## <a name="sharing-files-with-the-team"></a>Compartilhamento de arquivos com a equipe

Embora o armazenamento de arquivos e o gerenciamento de documentos seja um início, o compartilhamento com sua equipe e clientes externos também é importante. Os artigos a seguir ajudarão você a gerenciar usuários e compartilhar seus dados:
  
- Para compartilhamento de arquivos com sua equipe, consulte [Share SharePoint files or folders in Microsoft 365](https://support.microsoft.com/office/1fe37332-0f9a-4719-970e-d2578da4941c).
  
- Para entender o compartilhamento fora da sua equipe, consulte [Visão geral do compartilhamento externo.](/sharepoint/external-sharing-overview)
