# <a name="create-distribution-groups-in-the-microsoft-365-admin-center"></a>Criar grupos de distribuição no centro administrativo de Microsoft 365
  
Usados quando você deseja enviar e-mail para um grupo de pessoas sem ter que digitar o nome de cada destinatário individual, os grupos de distribuição (listas de distribuição) são organizados por um determinado assunto de discussão (como "Marketing") ou por usuários que compartilham trabalho comum que exige que eles se comuniquem com frequência. Elas são organizadas por um assunto específico (como "Marketing") ou por usuários que compartilham um trabalho comum que requer uma comunicação frequente.Também oferecem uma maneira de encaminhar emails automaticamente para vários endereços de email.

Grupos de distribuição às vezes são chamados de listas de distribuição.
  
## <a name="create-a-distribution-group-list"></a>Criar um grupo de distribuição (lista)

::: moniker range="o365-worldwide"

Vá para o centro de administração do<a href="https://go.microsoft.com/fwlink/p/?linkid=2024339" target="_blank">https://admin.microsoft.com</a>.

::: moniker-end

::: moniker range="o365-germany"

Vá para o centro de administração do<a href="https://go.microsoft.com/fwlink/p/?linkid=848041" target="_blank">https://portal.office.de/adminportal</a>.

::: moniker-end

::: moniker range="o365-21vianet"

Vá para o centro de administração do<a href="https://go.microsoft.com/fwlink/p/?linkid=850627" target="_blank">https://portal.partner.microsoftonline.cn/adminportal</a>.

::: moniker-end

1. Selecione o ícone do inicializador de aplicativos  e escolha **Administrador**.
    
    Não consegue encontrar o aplicativo que está procurando? No launcher do aplicativo, selecione **Todos os aplicativos** para ver uma lista alfabética dos aplicativos disponíveis para você. Nesse local, você pode pesquisar um aplicativo específico. 
    
2. Selecione  \> **Grupos no** painel de navegação esquerdo e, em seguida, selecione Adicionar um **grupo**. 
      
3. No **campo Escolher um tipo de grupo,** escolha **Distribuição** e escolha **o Próximo**.
  
4. Digite um nome, descrição e codinome para o seu novo grupo e escolha se deseja que as pessoas fora da sua organização enviem e-mails para o grupo. 
    
5. Pressione **Adicionar** para criar um grupo e, em seguida, revise seu grupo e escolha **Fechar**. 
    
6. Para adicionar usuários à sua lista de distribuição, consulte [Adicionar um usuário ou contato a um grupo de distribuição](../email/add-user-or-contact-to-distribution-list.md).
    
Confira como usar o grupo em Outlook 2016 e Outlook na web em [grupos de contato use (anteriormente listas de distribuição) em Outlook](https://support.microsoft.com/office/1c97fcb2-0ed4-41e6-b401-58f9d7d40e39). 
  
Confira [os problemas do grupo de distribuição](/office365/troubleshoot/groups/distribution-list-issues) para obter ajuda com os problemas da lista de distribuição.
