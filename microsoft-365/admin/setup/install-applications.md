# <a name="install-office-applications"></a>Instalar aplicativos do Office

Agora que você definiu o Microsoft 365, pode instalar aplicativos individuais do Office em seu Mac, computador ou dispositivos móveis.
  
Siga estes links para obter informações sobre como:
  
- Instalar aplicativos do Office:  [Instalar o Office em seu computador ou Mac](https://support.microsoft.com/office/4414eaaf-0478-48be-9c42-23adc4716658)

- Instalar outros aplicativos: [Project,](https://support.microsoft.com/office/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) [Visio](https://support.microsoft.com/office/install-visio-f98f21e3-aa02-4827-9167-ddab5b025710)ou [Skype for Business](https://support.microsoft.com/office/install-skype-for-business-8a0d4da8-9d58-44f9-9759-5c8f340cb3fb)

- Configurar dispositivos móveis: [Configuração móvel do Microsoft 365 - Ajuda](https://support.microsoft.com/office/7dabb6cb-0046-40b6-81fe-767e0b1f014f)

- Configurar email no Outlook: [Windows](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b) ou [Mac](https://support.microsoft.com/office/6e27792a-9267-4aa4-8bb6-c84ef146101b#PickTab=Outlook_for_Mac)
 
- [Atualizar os usuários para os aplicativos mais recentes](upgrade-users-to-latest-office-client.md) 

::: moniker range="o365-21vianet"

Se você comprou o Azure Active Directory Premium (AADP) Plano 1 ou Plano 2, está qualificado para o Microsoft Identity Manager (MIM). Para baixar o MIM, vá para a [Central de Download.](https://www.microsoft.com/zh-cn/download/details.aspx?id=58498)

::: moniker-end
