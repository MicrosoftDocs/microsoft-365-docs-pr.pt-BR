## <a name="upgrade-to-windows-10-pro"></a>Atualizar para o Windows 10 Pro
  
Para atualizar para o Windows 10 Pro, você tem várias opções. Você pode:
    
- Instale a atualização do site de Download de Software da [Microsoft](https://go.microsoft.com/fwlink/?LinkID=836951 ) Selecione essa opção se o dispositivo no qual você está conectado está no mesmo dispositivo que o que você &ndash; deseja atualizar. 

    - No site de download de software, clique em **Atualizar agora** para iniciar a atualização do dispositivo para Windows 10 Pro Creators Update. 
    
- Criar uma mídia [](https://go.microsoft.com/fwlink/?LinkID=836960) de instalação usando a Ferramenta de Criação de Mídia Selecione essa opção para criar uma mídia de instalação do Windows 10 Pro Creators Update (unidade flash USB ou arquivo ISO) para instalar o Windows 10 em um computador diferente do que você está &ndash; usando.

    - Leia as instruções sobre como usar a ferramenta e criar sua mídia de instalação. 

> [!NOTE]
> Se você tiver dispositivos Windows executando o Windows 7 Pro, Windows 8 Pro ou Windows 8.1 Pro, sua assinatura do Microsoft 365 para empresas lhe dará direito a uma atualização do Windows Pro 10.
    
Consulte [Configurar dispositivos Windows para o Microsoft 365 para](set-up-windows-devices.md) usuários comerciais para concluir a configuração de dispositivos Windows 10. 
  
Consulte [Configurar dispositivos móveis para o Microsoft 365 para](set-up-mobile-devices.md) usuários comerciais para concluir a configuração de dispositivos Android e iOS.
