# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Pré-requisitos para proteger dados em dispositivos com Microsoft 365 para empresas

Este artigo se aplica a Microsoft 365 Business Premium.

A primeira etapa para configurar sua organização com Microsoft 365 para empresas é garantir que você possa atender aos pré-requisitos.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Requisitos para configurar sua organização com o Microsoft 365 para empresas

- Windows dispositivos devem estar executando Windows 7 Professional, Windows 8 Pro ou Windows 8.1 Pro.
    
    [Atualizar dispositivos Windows para a Atualização do Windows Pro para Criadores](upgrade-to-windows-pro-creators-update.md)
    
    Se você estiver executando Windows 10 Home, deverá comprar **Windows** 10 Pro. Consulte [upgrade Windows 10 Home to Windows 10 Pro](../business-video/upgrade.md) para obter instruções. 
    
- Remova dispositivos das soluções de gerenciamento móvel (Mobile Iron, AirWatch e assim por diante). Você registrará todas as pessoas em sua organização no Microsoft 365 para gerenciamento móvel empresarial.
    
- Apple iOS 8.0 e versões posteriores.
    
    Google Android 4.0 e versões posteriores (incluindo Samsung KNOX Standard 4.0 e versões posteriores). Para obter mais informações, consulte [Intune supported devices](/mem/intune/fundamentals/supported-devices-browsers).
    
- Se você tiver aplicativos Office existentes em computadores de usuário, leia prepare-se para [Office](prepare-for-office-client-deployment.md) instalação do cliente para entender as etapas que você pode precisar seguir antes de configurar o Microsoft 365 para empresas para instalar o Office 2016 em computadores usuários.
