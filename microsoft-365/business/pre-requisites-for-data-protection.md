# <a name="prerequisites-for-protecting-data-on-devices-with-microsoft-365-for-business"></a>Pré-requisitos para proteger dados em dispositivos com o Microsoft 365 para empresas

Este artigo se aplica ao Microsoft 365 Business Premium.

A primeira etapa na configuração da sua organização com o Microsoft 365 para empresas é garantir que você possa atender aos pré-requisitos.
  
## <a name="requirements-for-setting-up-your-organization-with-microsoft-365-for-business"></a>Requisitos para configurar sua organização com o Microsoft 365 para empresas

- Os dispositivos Windows devem estar executando o Windows 7 Professional, Windows 8 Pro ou Windows 8.1 Pro.
    
    [Atualizar dispositivos Windows para a Atualização do Windows Pro para Criadores](upgrade-to-windows-pro-creators-update.md)
    
    Se você estiver executando o Windows 10 Home, deverá **comprar o** Windows 10 Pro. Consulte [atualizar o Windows 10 Home para o Windows 10 Pro](https://support.microsoft.com/office/0aee10c1-4d34-43ee-a325-579c6c2df90e) para obter instruções. 
    
- Remova dispositivos de soluções de gerenciamento móvel (Mobile Iron, AirWatch e assim por diante). Você registrará todas as pessoas em sua organização no gerenciamento móvel do Microsoft 365 para empresas.
    
- Apple iOS 8.0 e versões posteriores.
    
    Google Android 4.0 e versões posteriores (incluindo Samsung KNOX Standard 4.0 e versões posteriores). Para saber mais, confira [Dispositivos compatíveis com o Intune.](https://go.microsoft.com/fwlink/p/?linkid=852307)
    
- Se você tiver aplicativos existentes do Office em computadores do usuário, leia preparar-se para a instalação do cliente do [Office](prepare-for-office-client-deployment.md) para entender as etapas que talvez precise seguir antes de configurar o Microsoft 365 para empresas para instalar o Office 2016 nos computadores do usuário.
