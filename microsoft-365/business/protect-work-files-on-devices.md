# <a name="protect-work-files-on-devices"></a>Proteger arquivos de trabalho em dispositivos

No mundo atual, os usuários estão cada vez mais ligados aos dispositivos, e à medida que a vida pessoal e a profissional começam a se misturar, os dados da empresa podem acabar no dispositivo pessoal de um funcionário. As configurações que você define aqui podem ajudar a reassumir o controle e fornecem proteção para arquivos de trabalho, sem afetar os dados ou as configurações pessoais do usuário.
  
## <a name="configuring-policies-during-setup"></a>Configuração de políticas durante a instalação

Durante a instalação, três políticas de gerenciamento de aplicativo são adicionadas, sendo uma para cada dispositivo iOS, Android e Windows 10. As políticas têm configurações iguais e se aplicam a todos os usuários. As políticas são ativadas quando o usuário conecta a conta corporativa ao respectivo telefone celular.
  
Recomendamos aceitar a configuração padrão durante a instalação. Quando concluir a instalação, você poderá adicionar mais políticas que permitem ajustar a configuração e aplicar vários níveis de controle para grupos de usuários específicos.
  
Para adicionar políticas após a instalação, confira [Gerenciar políticas e dispositivos no Centro de administração](manage.md).
