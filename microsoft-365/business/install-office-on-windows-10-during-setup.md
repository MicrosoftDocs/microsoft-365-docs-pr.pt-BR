# <a name="install-office-on-windows-10-during-setup"></a>Instalar o Office para Windows 10 durante a configuração

![Faixa que aponta para https://aka.ms/aboutM365preview.](media/m365admincenterchanging.png)

Essa configuração é desmarcada por padrão. Você pode ativá-la para garantir que os usuários tenham a versão mais recente do Office em todos os seus dispositivos com Windows 10, mas devem primeiro verificar se todos os pré-requisitos foram atendidos. Os usuários devem ser licenciados para o Microsoft 365 Business para esta configuração enviar o software para o dispositivo, além de outros requisitos descritos em [Preparar para distribuição de cliente do Office com o Microsoft 365 Business](prepare-for-office-client-deployment.md). 
  
Recomendamos aceitar a configuração padrão durante a instalação. Caso prefira definir configurações diferentes para grupos de usuários diferentes, você poderá gerenciar a implantação por grupo após a instalação. Vá para **Dispositivos** \> **Gerenciar implantação do Office** na página inicial do Centro de administração. Confira as instruções passo a passo em [Instalar ou desinstalar o Office automaticamente em dispositivos Windows 10](auto-install-or-uninstall-office.md).
